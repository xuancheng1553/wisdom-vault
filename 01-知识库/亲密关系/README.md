# 亲密关系知识库

## 核心来源

### 罗兰·米勒《亲密关系》
系统整合人际关系科学研究成果的权威之作。

| 核心理论 | 一句话 |
|----------|--------|
| 吸引力法则 | 接近性、相似性、互惠性决定吸引力 |
| 依恋理论 | 安全型/焦虑型/回避型影响关系模式 |
| 奖赏理论 | 满意度 = 奖赏 - 成本 |
| 自我表露 | 逐步分享个人信息是亲密发展的核心机制 |

### 约翰·戈特曼《幸福的婚姻》
基于对上千对夫妻的长期观察的实证研究。

| 核心观点 | 一句话 |
|----------|--------|
| 爱情地图 | 深入了解伴侣的内心世界是婚姻基础 |
| 4个末日骑士 | 批评、鄙视、辩护、冷战是致命沟通方式 |
| 5:1比率 | 积极互动与消极互动至少5:1 |
| 69%问题无解 | 幸福夫妻学会与差异共处而非强行解决 |

### 盖瑞·查普曼《爱的五种语言》
- 肯定的言辞 - 精心的时刻 - 接受礼物 - 服务的行动 - 身体的接触

### 其他重要理论
- **约翰·鲍比** — 依恋理论（安全基地）
- **罗伯特·斯滕伯格** — 爱情三角理论（亲密+激情+承诺）
