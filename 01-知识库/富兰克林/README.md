# 本杰明·富兰克林知识库

## 核心贡献

### 十三条美德修身法
1. **节制** - 食不过饱，饮不过量
2. **沉默** - 言必有益，避免闲聊
3. **秩序** - 物归其位，事有定时
4. **决心** - 该做就做，做了就做好
5. **节俭** - 花钱于人己有益
6. **勤奋** - 珍惜时间，做有用的事
7. **诚实** - 不欺骗，不伤害他人
8. **正义** - 不损害他人利益
9. **中庸** - 避免极端，容忍他人
10. **整洁** - 保持身体、衣物和住所整洁
11. **平静** - 不为小事烦恼
12. **贞洁** - 慎行以保健康
13. **谦逊** - 效法耶稣和苏格拉底

### 推荐阅读
- 《富兰克林自传》
- 《穷查理年鉴》

### 与芒格的关联
芒格的名字叫查理，而富兰克林也写过《穷查理年鉴》。芒格一生推崇富兰克林，尤其是他的跨学科学习态度和实用主义哲学。
