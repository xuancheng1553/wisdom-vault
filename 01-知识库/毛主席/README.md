## 核心原则速览

| 原则 | 一句话 |
|------|--------|
| 实事求是 | 一切从实际出发，没有调查就没有发言权 |
| 矛盾论 | 矛盾是事物发展的根本动力，要抓主要矛盾 |
| 实践论 | 实践—认识—再实践—再认识，循环往复以至无穷 |
| 群众路线 | 从群众中来，到群众中去 |
| 独立自主 | 把方针放在自己力量的基点上 |
| 调查研究 | 没有调查就没有发言权，不做正确的调查同样没有发言权 |
| 战略藐视战术重视 | 敢于斗争，善于斗争 |
