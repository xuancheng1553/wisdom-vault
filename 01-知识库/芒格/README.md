## 核心原则速览

| 原则 | 一句话 |
|------|--------|
| 多元思维模型 | 手里有锤子，看什么都像钉子。必须掌握多学科的核心模型 |
| 反过来想 | 如果知道我会死在哪里，我就永远不去那里 |
| 能力圈 | 知道自己能力圈的边界，比能力圈本身更重要 |
| 双轨分析 | 理性分析 + 心理学偏误识别，双轨并行 |
| Lollapalooza效应 | 多个因素同向叠加产生的极端效应 |
| 耐心与等待 | 一生只做少数几个大决策，剩下的时间等待 |
| 检查清单 | 用清单避免决策中的遗漏和情绪干扰 |
