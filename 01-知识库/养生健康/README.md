# 养生健康知识库

## 核心来源

### 《黄帝内经》
中国最早的医学典籍，奠定了中医学的理论基础。

| 核心理念 | 阐释 |
|----------|------|
| 法于阴阳，和于术数 | 顺应自然规律生活 |
| 食饮有节，起居有常 | 饮食有节制、作息有规律 |
| 正气存内，邪不可干 | 提升免疫力是防病根本 |
| 上工治未病 | 预防大于治疗 |
| 四气调神 | 春生、夏长、秋收、冬藏 |
| 怒伤肝、喜伤心 | 情绪管理就是养生 |

### 孙思邈《千金要方》
唐代药王，活了141岁。

**养生十三法：** 发常梳、目常运、齿常叩、耳常鼓、面常洗、头常摇、腰常摆、腹常揉、膝常扭、脚常搓、常散步、常提肛、常深呼吸。

**四少口诀：** 口中言少、心中事少、腹里食少、自然睡少。依此四少，神仙诀了。

### 朱震亨（金元四大家）
阳常有余，阴常不足。养阴比补阳更重要。

## 养生原则速览

### 四季养生
| 季节 | 起居 | 养什么 |
|------|------|--------|
| 春 | 晚睡早起，多活动 | 养肝 |
| 夏 | 晚睡早起，不厌日 | 养心 |
| 秋 | 早睡早起，收敛神气 | 养肺 |
| 冬 | 早睡晚起，必待日光 | 养肾 |

### 日常养生
1. **饮食** — 七分饱、多样化、少生冷
2. **作息** — 跟着太阳走，子午觉要睡好
3. **运动** — 常欲小劳，莫大疲
4. **情志** — 情绪平和是最大的养生
5. **保暖** — 百病从脚起，脚暖全身暖
