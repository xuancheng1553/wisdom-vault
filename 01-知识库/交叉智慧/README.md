# 交叉智慧：芒格 × 毛主席

两位看似分属不同领域的思想家，在认知方法论上存在深刻共鸣。

## 对照维度

| 主题 | 芒格 | 毛主席 | 共通点 |
|------|------|--------|--------|
| 认识论 | 多元思维模型 | 实事求是 | 打破单一视角，从实际出发 |
| 方法论 | 反过来想 | 矛盾分析法 | 逆向思考、抓住本质 |
| 学习观 | 每天变聪明一点 | 活到老学到老 | 终身学习，理论与实践结合 |
| 战略观 | 耐心等待大机会 | 战略上藐视，战术上重视 | 长期主义与战术严谨 |
| 实践观 | 做对的艺术 | 实践是检验真理的唯一标准 | 强调动手、行动、验证 |
| 组织观 | 合伙人文化 | 民主集中制 | 信任与纪律的结合 |
