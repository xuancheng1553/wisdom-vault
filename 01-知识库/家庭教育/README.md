# 家庭教育知识库

## 核心来源

### 蒙特梭利教育法
强调儿童自主发展和有准备的环境。

| 核心理念 | 一句话 |
|----------|--------|
| 吸收性心智 | 0-6岁儿童通过环境自然吸收知识 |
| 敏感期 | 特定能力发展的最佳窗口期 |
| 有准备的环境 | 环境应适应儿童而非儿童适应环境 |
| 跟随儿童 | 观察而非强加，引导而非控制 |

### 简·尼尔森《正面管教》
核心理念：**和善而坚定**
- 既不要溺爱（和善不坚定）
- 也不要严厉（坚定不和善）
- 错误是学习的好机会
- 长期目标是培养自律、责任感、合作精神

### 其他重要理论

| 来源 | 核心价值 |

## 教育原则速览
1. **尊重儿童** — 把孩子当作独立的人来尊重
2. **环境优先** — 创造适合发展的环境比说教更有效
3. **言传身教** — 你想孩子成为什么样的人，你先成为
4. **和善坚定** — 爱与规则并行
5. **最近发展区** — 给予恰到好处的挑战和支持
6. **成长型思维** — 相信努力可以改变，从失败中学习
7. **自由与责任** — 给孩子选择的自由，也让他承担后果
8. **无条件的爱** — 爱不因为表现而改变
