# 曾国藩知识库

## 核心思想

### 修身十二条款
1. 主敬 - 整齐严肃，无时不惧
2. 静坐 - 每日静坐一小时
3. 早起 - 黎明即起，醒后不恋
4. 读书不二 - 一书未看完，不看他书
5. 读史 - 每日读史十页
6. 谨言 - 说话谨慎
7. 养气 - 无不可对人言之事
8. 保身 - 谨遵养生之道
9. 日知所亡 - 每日记下学到的新知识
10. 月无忘所能 - 每月复习已学
11. 作字 - 每日练字半小时
12. 夜不出门 - 晚上不外出应酬

### 家书核心思想
- 戒惰戒傲
- 读书可以变换气质
- 勤俭持家
- 孝友为家庭之祥瑞
