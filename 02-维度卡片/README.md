# 维度卡片

按人生十大维度，将芒格与毛主席的智慧映射到日常场景中。

## 维度一览

| 维度 | 芒格视角 | 毛主席视角 | 适用场景 |
|------|----------|------------|----------|
| 01-修身与学习 | 终身学习，多元思维 | 改造主观世界 | 个人成长 |
| 02-创业与事业 | 找到有护城河的生意 | 调查研究，抓住主要矛盾 | 创业决策 |
| 03-投资与理财 | 价值投资，能力圈 | 实事求是，权衡利弊 | 投资决策 |
| 04-婚姻与亲密关系 | 找到值得托付的伴侣 | 志同道合的革命伴侣 | 情感关系 |
| 05-社交与人脉 | 信任是最好的关系 | 团结一切可以团结的力量 | 人际关系 |
| 06-身体健康 | 长寿是复利的基础 | 身体是革命的本钱 | 健康管理 |
| 07-工作方法 | 检查清单，双轨分析 | 解剖麻雀，以点带面 | 日常工作 |
| 08-领导力与管理 | 以身作则，信任放权 | 民主集中，从群众中来 | 团队管理 |
| 09-齐家与教育 | 身教重于言传 | 家风建设 | 家庭经营 |
| 10-理想与境界 | 理性乐观主义 | 为人民服务 | 人生追求 |

每个维度的卡片文件将包含：
- 芒格相关智慧（引文 + 解读 + 当代应用）
- 毛主席相关智慧（引文 + 解读 + 当代应用）
- 交叉思考（两位智慧的结合应用）
- 行动建议（今天就能做的事）
