# 一页纸总结：四气调神

## 一句话定义
> 人的起居作息应该顺应四季的变化：春生、夏长、秋收、冬藏。

## 核心概念
1. **春三月**（立春到立夏）：晚睡早起，多活动，养肝
2. **夏三月**（立夏到立秋）：晚睡早起，不厌日，养心
3. **秋三月**（立秋到立冬）：早睡早起，收敛神气，养肺
4. **冬三月**（立冬到立春）：早睡晚起，必待日光，养肾

## 真实案例
- 春天不运动→夏天没精力
- 夏天过度吹空调→秋天容易感冒
- 秋天不收心→冬天容易焦虑
- 冬天熬夜→春天容易生病

## 常见错误
| 错误 | 表现 |
|------|------|
| 四季一样过 | 冬夏作息一致 |
| 反季节 | 冬天吃西瓜、夏天吃火锅 |
| 违背四气 | 春天克扣睡眠、冬天大量运动 |

## 检查清单
- [ ] 我知道现在是什么季节、该养什么？
- [ ] 我的作息符合现在的季节要求吗？
- [ ] 我的饮食符合季节吗？
- [ ] 我的运动量和季节匹配吗？
