# 一页纸总结：上善若水

## 一句话定义
> 最高的善就像水一样——滋养万物而不争功，甘居下位却无坚不摧。

## 核心概念
1. **利万物而不争**：创造价值但不居功自傲
2. **处众人之所恶**：甘愿做别人不愿意做的事
3. **柔能克刚**：水最柔软，却能穿透石头
4. **不争而胜**：不与人争，反而无人能与之争

## 真实案例
- 好的领导者：成就团队但不贪功（利万物而不争）
- 好的产品：解决用户痛点却不打扰用户（不争）
- 投资中的长期持有：不追涨杀跌（不争之德）

## 常见错误
| 错误 | 表现 |
|------|------|
| 把不争理解为不努力 | 消极躺平 |
| 把柔弱理解为软弱 | 不坚持原则 |
| 把不争功理解为不负责 | 该扛的不扛 |

- [ ] 我做事是为自己还是为价值？

- [ ] 我能接受功劳归别人、问题归自己吗？
