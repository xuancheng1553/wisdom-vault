# 一页纸总结：末日骑士

## 一句话定义
> 四种致命的沟通方式：批评、鄙视、辩护、冷战。它们的出现频率和顺序可以预测一段关系是否会破裂。

## 核心概念
1. **批评**：攻击人格而非具体行为（你总是这样！）
2. **鄙视**：侮辱、嘲讽、居高临下（你有什么用？）
3. **辩护**：为自己辩护而不倾听对方（不是我的错！）
4. **冷战**：拒绝沟通、筑墙撤退（沉默）

## 真实案例
批评：「你从来不洗碗」→ 正确：我希望能一起分担家务
鄙视翻白眼是最危险的信号——戈特曼说它可以预测离婚率87%

## 常见错误
| 错误 | 表现 |
|------|------|
| 批评当抱怨 | 你从来VS你能不能 |
| 不自知的鄙视 | 翻白眼、冷笑、啧 |
| 辩护等于反击 | 为自己辩护其实是攻击对方 |
| 冷战是逃避 | 不说话不等于解决问题 |

## 检查清单
- [ ] 我今天有没有用「你从来/你总是」开头说话？
- [ ] 我有没有对伴侣表现出不屑？
- [ ] 被批评时我是先听还是先辩解？
- [ ] 我有没有用沉默来逃避问题？
