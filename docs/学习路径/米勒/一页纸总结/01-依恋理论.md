# 一页纸总结：依恋理论

## 一句话定义
> 童年与照顾者形成的依恋模式会深刻影响成年的亲密关系。了解自己的依恋类型是改善关系的第一步。

## 核心概念
1. **安全型**：信任他人、不怕亲密、不担心被抛弃（约50%）
2. **焦虑型**：渴望亲密但害怕被抛弃、过度敏感（约20%）
3. **回避型**：害怕亲密、过度独立、不信任他人（约25%）
4. **混合型**：兼具焦虑和回避特征（约5%）

## 真实案例
- 焦虑型：TA怎么还不回微信？是不是不爱我了？
- 回避型：TA太粘人了，我需要空间
- 安全型：TA忙完会找我的，我先做自己的事

## 常见错误
| 错误 | 表现 |
|------|------|
| 贴标签 | 知道了类型就放弃改变 |
| 找错对象 | 焦虑型找回避型是互虐组合 |
| 宿命论 | 觉得依恋类型不能改变 |
