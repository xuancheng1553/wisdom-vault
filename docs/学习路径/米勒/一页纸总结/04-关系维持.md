# 一页纸总结：关系维持

## 一句话定义
> 好的关系不是没有问题的关系，而是双方持续投入、积极维护的关系。

## 核心概念
1. **五种维持策略**：积极性（乐观愉快）、开放性（自我表露）、保证（表达承诺）、社交网络（共同圈子）、共享任务（分担责任）
2. **5:1比率**：一次消极互动需要五次积极互动来抵消
3. **修复尝试**：冲突后主动示好是关系韧性的体现
4. **日常微互动**：问候、微笑、感谢——这些小事构建了关系的基础

## 真实案例
- ❌ 不做任何维护，等出了问题再补救 → 来不及了
- ✅ 每天做一些小事情：拥抱、夸一句、分担一件家务

## 常见错误
| 错误 | 表现 |
|------|------|
| 理所当然 | 觉得不用维护关系也应该好 |
| 只在节日表达 | 一年只在生日/纪念日表达爱 |
| 大棒政策 | 平时不维护，出问题用大礼物弥补 |
| 单方付出 | 只有一个人在维护关系 |

## 检查清单
- [ ] 今天有没有向伴侣表达欣赏？
- [ ] 今天有没有主动分享自己的想法？
- [ ] 最近有没有和伴侣一起做有趣的事？
- [ ] 我们有没有共同的朋友圈？
- [ ] 家务分工公平吗？

Q3：你的关系最需要哪种维持策略？
Q4：5:1比率在你们的关系中达标了吗？
Q5：修复尝试被对方识别和接受的关键是什么？
