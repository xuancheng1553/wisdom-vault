# 一页纸总结：关系解体

## 一句话定义
> 关系解体通常经历五个阶段：不满→讨论→决定→分离→结束。大多数人在第一阶段停留太久。

## 核心概念
1. **五个阶段**：不满→讨论→决定→分离→结束
2. **不满长期积累**：小问题不解决，积累到爆发时已无法挽回
3. **分手恢复过程**：不是线性的，是螺旋上升。今天好了明天又崩溃是正常的
4. **恢复策略**：允许悲伤→重构意义→重新投入→成长

## 真实案例
- 最常见的问题：不满积累期太长，错过了修复的最佳时机
- 分手后一周：痛不欲生→一个月：偶尔想起→三个月：开始释然→半年：成长了

## 常见错误
| 错误 | 表现 |
|------|------|
| 不沟通不满 | 忍到忍不了直接分手 |
| 分手后否认悲伤 | 压抑情绪反而恢复更慢 |
| 急于跳入新关系 | 不给自己时间消化和成长 |
| 重蹈覆辙 | 不反思就进入下一段，重复同样的模式 |
