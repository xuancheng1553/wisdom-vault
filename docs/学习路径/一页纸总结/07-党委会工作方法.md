# 一页纸总结：党委会的工作方法

## 一句话定义
> 十二条高效工作方法，核心是：透明沟通、统筹兼顾、心中有数。

## 十二条精华
1. **党委书记要善于当班长**：带头作用
2. **把问题摆到桌面上**：透明沟通
3. **互通情报**：信息共享
4. **不懂得和不了解的东西要问下级**：不懂就问，不装懂
5. **学会弹钢琴**：统筹兼顾，有主有次
6. **要抓紧**：定了的事一抓到底
7. **胸中有数**：用数据说话
8. **安民告示**：提前通知，不搞突击
9. **精兵简政**：会议简短、文件精简
10. **团结和自己意见不同的人**：包容异见
11. **力戒骄傲**：保持谦逊
12. **划清两种界限**：分清是非

## 真实案例
**会议管理**：
- ❌ 会前不发议程、会上发散讨论、会后没有结论
- ✅ 提前「安民告示」发议程、一个议题讨论完再过下一个（弹钢琴）、结束时明确结论和责任人（要抓紧）

**信息透明**：
- 每周开个短会互通信息，大家各自汇报进展和困难
- 发现问题及时「摆到桌面上」说清楚

## 常见错误
| 错误 | 表现 |
|------|------|
| 不透明 | 问题私下说，会上不说 |
| 抓而不紧 | 定了的事不跟踪 |
| 心中无数 | 凭感觉决策 |
| 搞突击 | 突然通知，不给准备时间 |

## 检查清单
- [ ] 最近开会前有发议程吗？
- [ ] 重要决定有没有记录和跟进？
- [ ] 团队成员间信息是否互通？
- [ ] 遇到问题是在桌面上说还是私下说？
- [ ] 我的决策有数据支撑吗？
- [ ] 我对不同意见保持开放吗？

## 5个自测题

**Q1：「学会弹钢琴」在工作中怎么应用？**
A：各项工作都要兼顾，但有主次节奏。本周最重要的事优先做，其他事穿插做。不能只顾一件不管其他，也不能平均用力。

**Q2：「要抓紧」为什么重要？**
A：定了不做等于没定。很多目标失败不是因为方向不对，而是因为没有坚持到底。
