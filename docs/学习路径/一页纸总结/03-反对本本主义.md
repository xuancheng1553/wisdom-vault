# 一页纸总结：反对本本主义

## 一句话定义
> 没有调查就没有发言权——一切结论必须建立在调查研究的基础上。

## 核心概念
1. **没有调查就没有发言权**：不了解情况就不要轻易下结论
2. **调查就是解决问题**：调研过程本身就是解决问题的一部分
3. **反对本本主义**：不要迷信书本、理论、上级指示，要从实际出发
4. **调查的技术**：开调查会、深入群众、抓住典型

## 真实案例
**创业案例**：想做社区团购
- ❌ 不走调查路线：看了一堆行业报告 → 照搬成功模式 → 发现本地用户不吃这套
- ✅ 正确路线：去社区和居民聊天 → 发现他们最关心的是生鲜品质 → 调整方向做品质路线

**管理案例**：团队士气低落
- ❌ 拍脑袋判断：「涨工资就好了」
- ✅ 调研后发现：大家最不满的是「做了事没人看到」，不是钱的问题

## 常见错误
| 错误 | 表现 | 后果 |
|------|------|------|
| 凭经验决策 | 「我以前就是这么做的」 | 用旧地图找新大陆 |
| 唯上唯书 | 「领导说的」「书上写的」 | 脱离实际 |
| 调查走过场 | 问几个人就下结论 | 样本偏差 |
| 带着结论调查 | 只找支持自己想法的证据 | 确认偏误 |

## 检查清单
- [ ] 这个结论是基于事实还是假设？
- [ ] 我亲自去了解过实际情况吗？
- [ ] 我的样本量够吗？有没有偏差？
- [ ] 我有没有先入为主的结论？
- [ ] 调查过程中有没有听到反面的声音？
- [ ] 我能列出3个不支持我当前想法的证据吗？

## 5个自测题

**Q1：「没有调查就没有发言权」和芒格的「能力圈」有什么共通之处？**
A：都是对自己认知边界的清醒认识——不了解的领域就不要指手画脚。

**Q2：「调查就是解决问题」是什么意思？**
A：调查不只是收集信息，在调查过程中你对问题的认识逐步深化，解决方案自然浮现。很多时候，认真调研完了，答案也出来了。

**Q3：你最近一次「不做调查就下结论」是什么时候？结果如何？**
（开放性题目，自己回答）

**Q4：调查中最容易犯的错误是什么？**
A：带着结论去调查，只找支持自己想法的证据（确认偏误）。好的调查应该去找反面证据。

**Q5：「反对本本主义」在今天的互联网时代有什么新含义？**
A：在信息过载的时代，更容易被各种「观点」和「理论」带着走。反对本本主义就是要回到事实本身。
