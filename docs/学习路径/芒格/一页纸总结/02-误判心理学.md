# 一页纸总结：误判心理学

## 一句话定义
> 系统地了解人类大脑常见的25个认知偏误，做决策时有意识地避开它们。

## 核心概念
1. **双轨分析**：理性分析轨道 + 心理偏误识别轨道，并行运行
2. **25个偏误**：激励机制、喜欢/热爱、讨厌/憎恨、避免怀疑、避免不一致、好奇心、公平、嫉妒、回馈、简单联想、否认、自视过高、过度乐观、损失厌恶、社会认同、对比误导、压力、易得性、不用就忘、化学物质、衰老、权威、废话、重视理由、Lollapalooza
3. **损失厌恶**：失去的痛苦远大于得到的快乐（2-3倍）
4. **Lollapalooza**：多个偏误同时作用，产生极端非理性

## 真实案例
- 投资中追涨杀跌：社会认同（别人都在买）+ 损失厌恶（怕错过）+ 过度乐观（我不会亏）
- 恋爱中被渣：喜欢/热爱（高估对方）+ 避免不一致（不想承认自己选错了）+ 简单联想（TA对我好过）

## 常见错误
| 错误 | 表现 |
|------|------|
| 知道但不识别 | 读过误判清单，但实际中认不出来 |
| 只识别他人 | 能看出别人犯错，看不到自己 |
| 过于自信 | 觉得「我不会犯这些错」 |
