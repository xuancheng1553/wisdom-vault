# 一页纸总结：实践论

## 一句话定义
> 实践是认识的来源，认识必须回到实践中检验，循环往复以至无穷。

## 核心概念
1. **感性认识**：通过五官接触外界获得的表面认识——看到、听到、感觉到
2. **理性认识**：经过思考加工，抓住事物本质和规律——分析、判断、推理
3. **再回到实践**：理性认识必须拿到实践中验证，检验它是否正确
4. **循环公式**：实践→认识→再实践→再认识→...（每循环一次深一层）

## 真实案例
**创业案例**：想做餐饮创业
- ❌ 错误做法：凭感觉觉得火锅赚钱 → 直接开店 → 3个月倒闭（跳过理性认识）
- ✅ 正确做法：先调研（感性认识）→ 分析竞品和用户需求（理性认识）→ 开个小摊验证（再实践）→ 成功再扩大

**投资案例**：想买一只股票
- ❌ 错误做法：听朋友推荐 → 全仓买入 → 跌了不知道怎么办
- ✅ 正确做法：读财报（感性）→ 分析护城河和估值（理性）→ 小仓位建仓（再实践）→ 验证后再加仓

## 常见错误
| 错误 | 表现 | 后果 |
|------|------|------|
| 经验主义 | 「我之前就是这样做成的」 | 用旧经验套新问题，忽视变化 |
| 教条主义 | 「书上/理论上是这么说的」 | 脱离实际，纸上谈兵 |
| 浅尝辄止 | 了解一点就以为全懂了 | 止于感性认识，看不到本质 |
| 只学不用 | 读了很多书但从不实践 | 永远停留在理论层面 |

## 检查清单
- [ ] 我对这个问题有**感性认识**吗？（亲自了解过吗？）
- [ ] 我做了足够的**调研**吗？（收集了多少数据？）
- [ ] 我能用一句话说清这个问题的**本质**吗？
- [ ] 我的结论需要在实践中**验证**吗？怎么验证？
- [ ] 验证周期是多久？失败了怎么办？
- [ ] 我是不是在**重复**这个循环？（还是停在某一步？）

## 5个自测题

**Q1：感性认识和理性认识的区别是什么？**
A：感性认识是表面的、片面的，如「这家店生意很好」；理性认识是本质的、全面的，如「这家店生意好是因为位置好+供应链强+复购率高」

**Q2：为什么理性认识必须回到实践中检验？**
A：因为理性认识只是「认为」是对的，是不是真的对，只有实践能回答。很多看起来很完美的理论，一做就发现错了。

**Q3：举一个你最近遇到的事情，说明你处于认识的哪个阶段？**
（开放性题目，自己回答）

**Q4：「读书是学习，使用也是学习，而且是更重要的学习」这句话怎么理解？**
A：读书获取的是别人的经验（间接认识），只有自己用了、在实践中验证了，才变成自己的真知。

**Q5：实践论的循环公式是什么？对你有何启发？**
A：实践→认识→再实践→再认识。启发：任何事不要指望一步到位，快速试错、快速调整才是正道。
