# 一页纸总结：富兰克林十三条美德

## 一句话定义
> 把修身分解为13个具体可执行的美德，每周专注一项，13周一个循环，一年循环4次。

## 核心概念
1. **系统化修身**：不是笼统地「做好人」，而是13个具体行为标准
2. **周循环法**：每周只盯一项，其他保持基本水准
3. **每日自省**：用表格记录每天是否践行，做不到就标记
4. **渐进改善**：不追求完美，追求持续进步

## 真实案例
- 富兰克林20岁开始践行，坚持了几十年
- 他并不完美，但他的方法让他在各个领域都取得了非凡成就
- 芒格借鉴了他的自我管理方法

## 常见错误
| 错误 | 表现 |
|------|------|
| 贪多 | 想同时改多个毛病 |
| 急于求成 | 一周没做到就放弃 |
| 完美主义 | 因为做不到完美就不做 |

## 检查清单
- [ ] 这周聚焦哪一项美德？
- [ ] 今天做到了吗？
- [ ] 这个循环（13周）比上个循环有进步吗？

## 5个自测题
Q1：为什么每周只做一项？A：大脑一次只能专注改变一个习惯。
Q2：十三条的顺序有讲究吗？A：有，从节制开始到谦逊，逻辑递进。
Q3：你当前最需要修炼的是哪一项？
Q4：和曾国藩的日课十二条有什么异同？
Q5：「不完美但持续」为什么比「完美但不持久」更好？
