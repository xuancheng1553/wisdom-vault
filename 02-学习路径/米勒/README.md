# 罗兰·米勒学习路径：5级进阶体系

## Level 1：认知入门
核心认知：《亲密关系》是人际关系科学最系统的教材。它不是鸡汤，是基于大量实证研究的科学结论。

## Level 2：方法掌握
- 吸引力法则：接近性、相似性、互惠性
- 依恋理论：安全型/焦虑型/回避型
- 奖赏理论：关系满意度 = 奖赏 - 成本
- 自我表露：逐步分享是亲密的核心
- 沟通模型：精确表述、积极倾听

## Level 3：实践应用
- **择偶**：识别安全型依恋
- **沟通**：用第一人称表达感受
- **冲突解决**：温和的开始、修复尝试
- **关系维护**：日常的积极互动

## Level 4：系统思维
- 亲密关系的科学发展史
- 依恋理论×沟通理论×奖赏理论的整合

## Level 5：融会贯通
- 米勒×戈特曼：学术研究×临床实证
- 米勒×孔子：己所不欲勿施于人×沟通理论
- 米勒×芒格：误判心理学×亲密关系偏误
