# 罗兰·米勒《亲密关系》学习路径：5级进阶体系

## Level 1：认知入门
> 目标：了解亲密关系科学的研究框架，破除常见迷思

| 维度 | 内容 |
|------|------|
| 核心认知 | 亲密关系不是直觉的，而是可以科学研究的 |
| 思想地图 | 吸引力→认知→沟通→依赖→爱情→冲突→解体→维持 |
| 常见迷思 | 真爱可以克服一切、吵架就是关系不好、好的关系不需要经营 |

## Level 2：方法掌握
> 目标：掌握核心理论框架

- 吸引力法则（接近性、相似性、互惠性）
- 依恋理论（安全/焦虑/回避）
- 社会认知（信念、归因、期望）
- 相互依赖理论（奖赏-成本-承诺）
- 爱情三角理论（亲密-激情-承诺）
- 关系维持策略（五种方法）

## Level 3：实践应用
> 目标：把理论应用到择偶、沟通、冲突解决

- **择偶**：识别安全型依恋、识别自我服务偏差
- **沟通**：第一人称表达、温和开始、修复尝试
- **冲突**：四步法、要求-退避模式的破解
- **日常**：五种关系维持策略

## Level 4：系统思维
> 目标：建立亲密关系的系统认知

- 各章之间的内在逻辑链
- 从吸引力到解体的完整生命周期
- 多个理论框架的交叉验证

## Level 5：融会贯通
> 目标：将亲密关系科学与其他智慧交叉

- 米勒×戈特曼：学术研究×临床实证
- 米勒×芒格：误判心理学×关系认知偏差
- 米勒×孔子：己所不欲×沟通艺术
