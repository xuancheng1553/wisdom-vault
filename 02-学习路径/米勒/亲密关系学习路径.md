# 亲密关系5级学习路径

> 基于罗兰·米勒《亲密关系》14章体系，拆解为5个可执行的等级。
> 每级含：掌握内容 + 常见错误 + 进阶标准 + 练习

---

## Level 1：认知入门——亲密关系到底是什么

### 掌握内容
- 亲密关系的7个特征：了解、关心、相互依赖、相互一致、信任、承诺、排他
- 为什么亲密关系可以科学研究——不是玄学
- 吸引力的三大基石：接近性、相似性、互惠性
- 依恋理论的三种类型：安全型、焦虑型、回避型
- 14章的内容地图：吸引力→认知→沟通→依赖→爱情→冲突→解体→维持

### 推荐阅读
- 《亲密关系》第1章（导论）、第3章（吸引力）、第6章（依恋理论部分）

### 练习
- 完成依恋类型自测（网上有免费量表）
- 写下自己对亲密关系的核心困惑（至少3个问题）
- 回忆一段关系，分析其中的吸引力三要素

### 常见错误
| 错误 | 表现 |
|------|------|
| 把亲密关系当玄学 | 觉得爱情不可分析，不需要学 |
| 只看结论不学原理 | 知道依恋类型就贴标签，不了解背后的机制 |
| 宿命论 | 觉得「我就是焦虑型，改不了」 |
| 只看符合自己的部分 | 只关注自己认同的观点 |

### 进阶标准
- 能说出亲密关系的7个特征
- 知道自己的依恋类型
- 能用吸引力三要素分析一段关系
- 读完第1、3、6章

---

## Level 2：方法掌握——核心理论框架

### 掌握内容
- **社会认知**：关系信念（成长型vs宿命型）、归因方式（正面归因vs负面归因）
- **相互依赖理论**：奖赏-成本-承诺模型、比较水平、替代选择
- **爱情三角理论**：亲密、激情、承诺三个成分及七种爱情类型
- **沟通模型**：精确表述、积极倾听、要求/退避模式、修复尝试

### 推荐阅读
- 《亲密关系》第4章（社会认知）、第6章（相互依赖）、第8章（爱情）、第5章（沟通）

### 练习
- 连续3天记录自己对伴侣/潜在对象的归因方式（好的是人品还是运气？坏的是性格还是情境？）
- 用爱情三角理论分析自己的关系，看缺了什么成分
- 用第一人称表达一次感受（代替指责）

### 常见错误
| 错误 | 表现 |
|------|------|
| 混合不同理论 | 把爱情三角和相互依赖混在一起乱用 |
| 只学一种理论 | 只用依恋理论解释一切，忽视其他因素 |
| 归因偏差 | 好的归自己、坏的归对方 |
| 知道了但不改变 | 知道归因方式有问题，但不调整 |

### 进阶标准
- 能用2个以上理论分析同一段关系
- 能识别自己的归因偏差
- 能在沟通中使用第一人称表达
- 读完第4、5、6、8章

---

## Level 3：实践应用——把理论用起来

### 掌握内容
- **冲突解决**：冲突四步法（冷静→共情→表达→方案）、要求/退避模式的破解
- **关系维持**：五种维持策略（积极性、开放性、保证、社交网络、共享任务）
- **性亲密**：性与关系满意度的相互影响、性欲不匹配的处理
- **压力管理**：嫉妒的类型与处理、欺骗与信任修复

### 推荐阅读
- 《亲密关系》第9章（性）、第10章（压力与紧张）、第11章（冲突）、第14章（维持与修复）
- 《幸福的婚姻》（戈特曼）第3-8章

### 练习
- 下次吵架时用「冲突四步法」走一遍
- 用五种维持策略评估自己的关系，找出最需要改进的一项
- 一周内刻意增加积极互动（每天至少1次）
- 练习温和地开启一场困难对话

### 常见错误
| 错误 | 表现 |
|------|------|
| 理论背得熟但吵架时全忘了 | 练习不够，关键时刻用不出来 |
| 只要求对方改变 | 「你应该怎么怎么样」 |
| 一次性用太多技巧 | 显得不自然，对方觉得你在「套路」 |
| 忽视修复 | 吵完就算，不修复情感连接 |

### 进阶标准
- 能在关系中有意识地使用沟通技巧
- 掌握冲突四步法并成功使用过
- 制定了关系维持计划并执行了2周
- 读完第9、10、11、14章

## Level 4：系统思维——打通各章之间的联系

### 掌握内容
- **理论整合**：吸引力→社会认知→沟通→相互依赖→爱情→冲突→解体→维持的完整生命线
- **交叉验证**：同时用依恋理论 + 社会认知 + 相互依赖分析同一个问题
- **深度洞察**：理解为什么焦虑型和回避型互相吸引、为什么日常忽视比重大冲突更致命

### 推荐阅读
- 《亲密关系》全书回顾
- 《依恋》阿米尔·莱文（依恋理论的实际应用）

### 练习
- 画出亲密关系的「生命周期图」（从吸引到维持或解体）
- 用3个理论同时分析自己的一段重要关系
- 写一篇2000字的关系分析笔记

### 常见错误
| 错误 | 表现 |
|------|------|
| 只用单一理论 | 拿一个锤子看所有钉子 |
| 忽略时间维度 | 关系是动态变化的，不同阶段核心矛盾不同 |
| 过度分析 | 把关系当成解剖实验，失去了体验 |
| 忽视自己 | 分析来分析去，不行动 |

### 进阶标准
- 能在一段关系分析中综合使用3个以上理论
- 能从生命周期角度理解关系的变化
- 写出一篇系统性的关系分析笔记

---

## Level 5：融会贯通——形成自己的关系哲学

### 掌握内容
- **米勒×戈特曼**：学术研究与临床实证的结合
- **米勒×依恋理论**：从理论到实践的延伸
- **米勒×沟通理论**：为什么第一人称表达如此有效
- **形成个人关系哲学**：你的价值观、原则、底线

### 推荐阅读
- 米勒《亲密关系》全书 + 戈特曼《幸福的婚姻》
- 查普曼《爱的五种语言》
- 约翰逊《依恋与亲密关系》（EFT疗法）

### 练习
- 写一份自己的「关系原则清单」（什么是可以容忍的、什么是不能接受的）
- 写一篇3000字的亲密关系哲学陈述
- 把学到的教给别人——教是最好的学

### 常见错误
| 错误 | 表现 |
|------|------|
| 套用他人哲学 | 照抄别人的原则，没有自己消化 |
| 纸上谈兵 | 理论一套套，现实中该犯的错照犯 |
| 忽视个人差异 | 认为所有关系都适用同一套规则 |
| 完美主义 | 追求完美的关系，不切实际 |

### 进阶标准
- 形成了自己的关系哲学框架
- 能把学到的知识教给别人
- 在实践中能灵活运用理论，不僵化
- 建立了持续学习和反思的习惯
