# 一页纸总结：吸引力

## 一句话定义
> 人与人之间产生吸引力的基本假设是：他人的出现对我们有奖赏意义。接近性、相似性、互惠性是三大基石。

## 核心概念
1. **接近性**：物理距离近的人更容易产生好感（见面机会多）
2. **相似性**：态度、价值观、兴趣相似的人互相吸引——物以类聚
3. **互惠性**：你喜欢我，所以我也喜欢你——好感是相互的
4. **吸引力奖赏理论**：他人带给我们的奖赏（愉悦/支持/资源）决定了吸引程度
5. **匹配现象**：人们倾向于选择和自己整体吸引力相当的人

## 真实案例
- 为什么办公室恋情多？——接近性
- 为什么相亲时聊得来很重要？——相似性
- 为什么对方对你有好感你就更容易喜欢TA？——互惠性

## 常见错误
| 错误 | 表现 |
|------|------|
| 误把接近当喜欢 | 只是常见面就以为是真爱 |
| 忽略价值观相似 | 觉得性格互补最重要，价值观不同无所谓 |
| 单向追逐 | 单方面讨好不会换来真心 |

## 检查清单
- [ ] 我和TA的接近性是暂时的还是可以持续的？
- [ ] 我们的价值观是否真正匹配？
- [ ] 好感是相互的还是单向的？
- [ ] 我们整体吸引力水平是否匹配？
