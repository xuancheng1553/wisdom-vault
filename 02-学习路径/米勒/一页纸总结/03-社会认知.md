# 一页纸总结：社会认知

## 一句话定义
> 我们对伴侣和关系的看法、信念、归因方式，比客观事实更能决定关系满意度。

## 核心概念
1. **关系信念**：成长信念（关系可以改善）vs 宿命信念（要么合适要么不合适）
2. **归因方式**：幸福夫妻把好的归因于人品，把坏的归因于情境
3. **自我服务偏差**：高估自己的贡献，低估对方的贡献
4. **期望效应**：你期望什么，往往就会得到什么

## 真实案例
- ❌ 宿命信念者：我们就是不合适 → 放弃努力 → 关系真的变差
- ✅ 成长信念者：我们遇到了问题 → 一起解决 → 关系变得更强

## 常见错误
| 错误 | 表现 |
|------|------|
| 负面归因 | TA迟到是因为不在乎（归因人品） |
| 正面归因偏差 | 我迟到是因为堵车（归因外部） |
| 确认偏误 | 只找支持自己看法的证据 |
| 宿命信念 | 觉得好的关系不需要努力 |

## 检查清单
- [ ] TA做了一件好事，我归因于内在还是外在？
- [ ] TA做了一件不好的事，我归因于内在还是外在？
- [ ] 我最近有没有高估自己的贡献？
- [ ] 我持有成长信念还是宿命信念？
- [ ] 我对这段关系的期望合理吗？
