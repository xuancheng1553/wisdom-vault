# 黄帝内经10次×2小时学习计划

> 基于《素问》核心篇章 + 《灵枢》关键概念
> 每次2小时 = 阅读 + 练习 + 思考 + 复盘
> 每日微任务：5-10分钟，两课之间每天做

---

## 第1次：认识黄帝内经——一本不是医书的生命哲学

**阅读**（45min）
- 《黄帝内经》成书背景、篇章结构（素问+灵枢）
- 《素问·上古天真论》开篇：黄帝问天师岐伯
- 核心段落："上古之人，其知道者，法于阴阳，和于术数，食饮有节，起居有常，不妄作劳，故能形与神俱，而尽终其天年，度百岁乃去。"

**练习**（30min）
- 对照上古之人的养生原则，给自己打分（每项1-10分）：
  - 法于阴阳（顺应自然）：__分
  - 和于术数（运动适度）：__分
  - 食饮有节（饮食有度）：__分
  - 起居有常（作息规律）：__分
  - 不妄作劳（不过度消耗）：__分
- 写出你最需要改进的一项

**思考**（30min）
- 为什么说黄帝内经不是医书而是一套生命哲学？
- 你在读开篇时，哪个词最打动你？
- 现在的"养生焦虑"和内经的"顺应自然"是什么关系？

**复盘**（15min）
- 今天学到的最反直觉的一个观点是什么？
- 如果只能带走一个观念，你带走哪一个？

**每日微任务**（每天5min）
- Day1-2：观察自己一天中有多少决策是"顺应自然"的？
- Day3-4：记录你的饮食和睡眠，找出1个违反养生原则的习惯
- Day5-7：每天读一段上古天真论，并写一句感悟

## 第2次：四气调神——跟着四季过日子

**阅读**（45min）
- 《素问·四气调神大论》全文
- 重点：春生、夏长、秋收、冬藏的具体做法
- "夫四时阴阳者，万物之根本也。所以圣人春夏养阳，秋冬养阴，以从其根"

**练习**（30min）
- 判断现在是什么季节（包含长夏），该养什么？
- 对照当前季节的要求，列出你在作息/饮食/运动上的3个差距
- 制定"本季节养生计划"（30天可执行版）

**思考**（30min）
- 为什么春天要"生而勿杀"？这和"生机"的关系是什么？
- 夏天吹空调、冬天吃西瓜——这些现代便利反季节行为有什么影响？
- 四季养生的底层逻辑是什么？

**复盘**（15min）
- 四季养生的核心是"顺应"二字——你今天对"顺应"有什么新理解？
- 你的季节养生计划能执行下去吗？最大障碍是什么？

**每日微任务**（每天5-10min）
- Day1-7：执行你的"本季节养生计划"第一周
- 每天注意：今天的运动/饮食/作息是否顺应了季节？

## 第3次：阴阳应象——理解阴阳的底层逻辑

**阅读**（45min）
- 《素问·阴阳应象大论》核心段落
- 阴阳的基本概念：对立统一、消长转化、互根互用
- "阴阳者，天地之道也，万物之纲纪，变化之父母，生杀之本始，神明之府也"

**练习**（30min）
- 画出阴阳太极图并标注：阴阳的互根、消长、转化
- 用阴阳分析生活中的一个健康问题（如：失眠=阳不入阴）
- 列出你身体上"阴阳不平衡"的表现（怕冷/怕热/口干/水肿等）

**思考**（30min）
- 为什么说"阴平阳秘，精神乃治"——平衡比补什么更重要？
- 阴阳理论和芒格的"双轨分析"有什么相似之处？
- 你现在的身体状态是阳偏盛还是阴偏虚？怎么判断？

**复盘**（15min）
- 阴和阳不是两个东西，是一个事物的两面——你能举一个生活例子吗？
- 今天学到的阴阳知识，明天就能用的一个点是什么？

**每日微任务**（每天5-10min）
- Day1：观察一天中的阴阳变化：上午为阳、下午为阴
- Day2-3：识别自己的"阴虚"或"阳虚"信号
- Day4-5：找一个身边阴阳不平衡的例子（如：熬夜后第二天的状态）
- Day6-7：读《阴阳应象大论》选段，写一句心得

## 第4次：五行生克——理解身体的能量网络

**阅读**（45min）
- 五行理论的基本框架：木火土金水
- 五行的相生关系：木生火、火生土、土生金、金生水、水生木
- 五行的相克关系：木克土、火克金、土克水、金克木、水克火
- 五行对应五脏：肝（木）、心（火）、脾（土）、肺（金）、肾（水）

**练习**（30min）
- 画出五行生克图，标出五脏对应关系
- 用五行分析一个健康问题：比如长期忧虑（伤脾土）→胃口不好→导致气血不足（金不生水）
- 找出你自己最弱的"一行"（哪个脏腑最容易出问题？）

**思考**（30min）
- 五行为什么是"相生"又"相克"？这不矛盾吗？
- "病传"的意思是什么？一个脏出问题会传到哪里？
- 五行理论和现代医学的"系统关联"有什么异曲同工？

**复盘**（15min）
- 五行不是命理，是功能模型——你今天理解了吗？
- 你的五脏最需要关注的是哪个？

**每日微任务**（每天5-10min）
- Day1：背下五行-五脏-五季-五色对应表
- Day2-3：观察你吃的食物颜色，想想对应哪个脏
- Day4-5：识别一个情绪对应哪个脏腑（如发怒对应肝）
- Day6-7：画一张自己的"五行健康图"

## 第5次：藏象学说——五脏六腑不是解剖学

**阅读**（45min）
- 《素问·灵兰秘典论》全文
- "心者，君主之官也，神明出焉"——五脏的功能定位
- 六腑功能：胆、胃、大肠、小肠、膀胱、三焦

**练习**（30min）
- 画出五脏的功能关系图，标出"君臣佐使"的层级
- 对照自己的症状，识别哪个脏在"报警"：
  - 失眠多梦？→ 心
  - 眼睛干涩？→ 肝
  - 胃口不好？→ 脾
  - 容易咳嗽？→ 肺
  - 腰酸怕冷？→ 肾

**思考**（30min）
- 为什么中医说"心主神明"而西医说心脏只是泵？两种医学怎么理解这个差异？
- 藏象学说对我们理解"情绪-器官"的关联有什么启发？
- 你最能感同身受的五脏关系是哪个？

**复盘**（15min）
- 藏象不是解剖，是功能——这个理解到位了吗？
- 你的哪个脏最需要养护？

**每日微任务**（每天5-10min）
- Day1-2：背下五脏对应的五华（心在面/肝在目/脾在唇/肺在鼻/肾在耳）
- Day3-4：每天照镜子看看面色/舌苔
- Day5-7：记录一周的身体信号，对照五脏分析

## 第6次：经络学说——人体的能量网络

**阅读**（45min）
- 《灵枢·经脉》核心内容
- 十二正经：手三阴、手三阳、足三阴、足三阳
- 任督二脉（奇经八脉中最重要）
- "经脉者，所以能决死生，处百病，调虚实，不可不通"

**练习**（30min）
- 在自己的身体上标出3条主要经络的走向（如：胃经走面部→前胸→腿前侧）
- 找到你最常疼痛的部位，判断是哪个经的问题
- 学会按揉2个常用保健穴位：足三里（胃经）和涌泉穴（肾经）

**思考**（30min）
- 为什么说"通则不痛，痛则不通"？
- 经络和神经/血管/淋巴的关系是什么？
- 你的哪个经络最容易瘀堵？怎么判断？

**复盘**（15min）
- 经络学说的核心是什么？
- 今天按揉穴位的感觉如何？有酸麻胀痛吗？

**每日微任务**（每天5-10min）
- Day1-7：每天睡前按揉足三里3分钟 + 涌泉穴3分钟
- 每天拍打大腿内侧（肝经）100下
- 记录身体感受的变化

## 第7次：饮食养生——食饮有节的智慧

**阅读**（45min）
- 《素问·藏气法时论》关于饮食的论述
- "五谷为养、五果为助、五畜为益、五菜为充"的搭配原则
- 五味与五脏的关系：酸入肝、苦入心、甘入脾、辛入肺、咸入肾

**练习**（30min）
- 记录今天吃的东西，按"五谷/五果/五畜/五菜"分类
- 根据当前季节，制定一周的应季食谱
- 检查自己在"七分饱"上能做到吗？记录下来

**思考**（30min）
- "食饮有节"——节是节制还是节奏？为什么？
- 补品/保健品和前人的"五谷为养"矛盾吗？
- 你最大的饮食问题是什么？（吃太多？吃太晚？吃太凉？）

**复盘**（15min）
- 饮食养生的核心原则是什么？
- 明天开始你能改变的一个饮食坏习惯是什么？

**每日微任务**（每天5-10min）
- Day1-7：每天记录一次饮食，标注"五谷/五果/五畜/五菜"占比
- 每餐前问自己：七分饱了吗？
- 吃到应季食材时特别感受一下

## 第8次：情志养生——情绪就是身体的信号

**阅读**（45min）
- 《素问·举痛论》关于"百病生于气"的论述
- 怒伤肝、喜伤心、思伤脾、忧伤肺、恐伤肾的五志关联
- "恬淡虚无，真气从之；精神内守，病安从来"

**练习**（30min）
- 记录今天经历过的所有明显情绪，标注对应的受伤脏腑
- 找出你最频繁出现的情绪问题（易怒/焦虑/忧虑/恐惧？）
- 针对这个情绪，设计一个"5分钟快速调节法"

**思考**（30min）
- 为什么情绪会伤身？心理影响生理的机制是什么？
- "喜伤心"——难道开心也不好吗？那什么是"适度"？
- 情志管理的最高境界是"恬淡虚无"——你愿意过这样的生活吗？

**复盘**（15min）
- 你最容易伤到哪个脏？对应的情绪是什么？
- 你今天用了一个什么方法来调节情绪？

**每日微任务**（每天5-10min）
- Day1-7：每天睡前做"情绪复盘"——今天什么情绪最多？对应的脏是哪个？
- 烦躁时深呼吸10次（调气）
- 愤怒时按揉太冲穴（肝经原穴）

## 第9次：治未病——最高明的医生是让自己不生病的医生

**阅读**（45min）
- 《素问·四气调神大论》关于"治未病"的论述
- "是故圣人不治已病治未病，不治已乱治未乱"
- 预防医学的三个层级：未病先防、既病防变、瘥后防复

**练习**（30min）
- 评估自己的"未病状态"：哪些小毛病在反复出现却还没发展成大病？
- 制定一个"90天未病预防计划"：
  - 饮食预防：戒掉什么
  - 作息预防：调整什么
  - 运动预防：增加什么
  - 情志预防：管理什么
- 对标你之前的身体检查报告，哪些指标是需要"未病先防"的？

**思考**（30min）
- 为什么说"上工治未病"——最高明的医生是预防的医生？
- 现在的体检制度和治未病理念的关系是什么？体检能代替治未病吗？
- 你过去有没有"小病不治变成大病"的经历？

**复盘**（15min）
- 治未病的三个层级你理解了吗？你处在哪个层级？
- 你制定的预防计划中，最关键的一项是什么？

**每日微任务**（每天5-10min）
- Day1-7：执行你的90天未病预防计划第一周
- 每天记录一个"今天做对了/做错了"的养生决策

## 第10次：融会贯通——形成自己的健康哲学

**阅读**（45min）
- 回顾前9次的所有笔记和练习记录
- 重读上古天真论和四气调神大论两篇核心原文
- 读一段对比材料：内经养生×现代科学（生物钟/自愈力/慢性炎症）

**练习**（30min）
- 写出你的"个人健康哲学宣言"（200-300字）：
  - 我的养生核心理念
  - 我最坚持的3个健康习惯
  - 我最警惕的3个健康风险
- 画一张"我的人体系统图"——标出你最需要关注的部分
- 制定"年度养生计划"：四季各有重点

**思考**（30min）
- 学完黄帝内经后，你对"健康"的定义有什么变化？
- 哪一条内经智慧对你触动最深？为什么？
- 你觉得中医养生和现代科学最终会走向融合吗？

**复盘**（15min）
- 10次学习中最有价值的3个收获是什么？
- 你最需要坚持的一个内经原则是什么？

**每日微任务**（每天5-10min）
- Day1：把你的健康哲学宣言贴在能看到的地方
- Day2-3：开始执行年度养生计划
- Day4-5：把这个月养生的心得写下来
- Day6-7：复盘并调整计划

## 日常养生微习惯（两课之间）

| 时间 | 微习惯 | 内经依据 |
|------|--------|----------|
| 晨起 | 喝温水、伸懒腰、不立刻看手机 | "春气之应，养生之道" |
| 午时（11-13点） | 小憩15-30分钟 | 心经当令，养心 |
| 下午 | 走动5分钟，不连续久坐 | "久坐伤肉" |
| 晚餐 | 七分饱、不吃太晚 | "食饮有节" |
| 子时前（23点前） | 上床睡觉 | "冬气之应，养藏之道" |
| 睡前 | 泡脚10分钟 | 引火归元，助眠 |
| 日常 | 察觉情绪，不过度压抑或发泄 | "恬淡虚无，真气从之" |
| 每周 | 一次适度运动（散步/太极/八段锦） | "常欲小劳莫大疲" |
