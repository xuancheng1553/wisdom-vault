# 黄帝内经学习路径：5级进阶体系

## 设计理念
《黄帝内经》不是一本医书，而是一套关于「人如何与自然和谐相处」的生命哲学体系。它的核心不是治病，是**不生病**——通过顺应自然规律来保持身体和精神的平衡。这套路径从"认知"到"融通"，让你真正用内经开智慧，而不是当一本老古董来背。

---

## Level 1：认知入门
> 目标：了解黄帝内经是什么、不是医书是生命哲学、核心思想地图

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 核心认知 | 《黄帝内经》分《素问》81篇和《灵枢》81篇，前者讲生命规律，后者讲针灸经络 | 能用一句话说清"它是什么" |
| 思想地图 | 天人合一、阴阳平衡、五行生克、藏象功能、经络气血、治未病 | 能说出6个核心概念 |
| 阅读起点 | 《素问·上古天真论》开篇：黄帝问"上古之人为何能活百岁" | 读完开篇并做笔记 |
| 常见错误 | 把内经当药方书（它不是）、觉得是玄学（它有严谨体系）、只看不练 | 避免三大入门误区 |
| 进阶标准 | 对"顺应自然"有了体感，能说出自己生活中最需要调整的一个习惯 | 读完上古天真论 |

## Level 2：方法掌握
> 目标：掌握阴阳、五行、藏象、经络等核心概念

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 阴阳理论 | 对立统一、消长转化、互根互用——万物皆分阴阳 | 能用阴阳分析生活中的健康问题 |
| 五行生克 | 木火土金水——相生相克、相乘相侮的关系网络 | 能画出五行生克图并举例 |
| 藏象学说 | 五脏（心肝脾肺肾）的功能不只是解剖，更是系统功能 | 能说出五脏对应的功能+季节+情绪 |
| 经络基础 | 十二正经+奇经八脉——人体的能量通道网 | 能说出3条主要经络的走向和功能 |
| 四气调神 | 春生夏长秋收冬藏——四季养生的具体方法 | 能制定本季度的养生计划 |
| 常见错误 | 把阴阳当玄学、把五行当迷信、用西医解剖对标中医藏象 | 避免四大理解误区 |
| 进阶标准 | 能用自己的话解释阴阳五行+说出自己属于什么体质 | 建立自己的体质认知 |

## Level 3：实践应用
> 目标：把内经智慧用在饮食、作息、运动、情志四个方面

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 饮食 | 食饮有节、七分饱、五谷为养、顺应四时吃当季食物 | 能制定一周的顺应季节的食谱 |
| 作息 | 跟着太阳走、子午觉（子时前睡、午时小憩）、春夏晚睡早起秋冬早睡晚起 | 能调整作息符合四季规律 |
| 运动 | 常欲小劳莫大疲——最好的运动是"不过度"，如散步、太极、八段锦 | 能制定适合自己的运动方案 |
| 情志 | 怒伤肝、喜伤心、思伤脾、忧伤肺、恐伤肾——情绪过激就是病 | 能识别自己最需要调节的情绪 |
| 常见错误 | 照搬古法不适应现代生活（如古代日出而作）、过度追求"养生"变成焦虑 | 避免现代应用的两大陷阱 |
| 进阶标准 | 在2个以上生活场景中实践内经原则并看到改善 | 建立个人养生日志 |

## Level 4：系统思维
> 目标：打通藏象、经络、阴阳、五行的整体网络，理解"整体观"和"辨证论治"

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 整体观 | 人体是一个小宇宙——内外相通、五脏相联、身心一体 | 能画出人体各系统的关联图 |
| 辨证论治 | 同一个病、不同人不同治法——因人因时因地制宜 | 能说出一两个"同病异治"的案例 |
| 气血津液 | 气是动力、血是营养、津液是润泽——三者失衡就是病 | 能判断自己是气虚/血虚/湿气重 |
| 体质辨识 | 阳虚、阴虚、气虚、湿热、痰湿、血瘀等九种体质 | 能说出自己体质+日常注意 |
| 常见错误 | 只调一处不管整体、只用一种方法（只吃不吃动）、忽略个体差异 | 形成系统思维 |
| 进阶标准 | 能从"整体观"去理解自己和家人的健康问题 | 能做一次完整的身体状态分析 |

## Level 5：融会贯通
> 目标：将内经智慧与其他智慧体系交叉印证，形成自己的健康哲学

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 内经×芒格 | 整体观×多元思维模型、取象比类×模型思考、治未病×预防思维 | 写一篇对比分析 |
| 内经×纳瓦尔 | 健康是一切基础×养生是健康基石、活在当下×顺四时 | 写一篇对比分析 |
| 内经×老子 | 道法自然×天人合一、无为×不妄作劳 | 写一篇对比分析 |
| 内经×稻盛和夫 | 工作即修行×身体是修行的基础、情志管理×不要有感性的烦恼 | 写一篇对比分析 |
| 内经×现代科学 | 循证思维×中医理论验证、昼夜节律×生物学中的生物钟 | 写一篇对比分析 |
| 常见错误 | 用西医标准衡量中医（是不同的体系）、把内经当唯一真理（要与时俱进）、理论和实践脱节 | 形成自己的健康哲学 |
| 进阶标准 | 形成了自己的健康哲学体系，能兼顾传统智慧和现代科学 | 建立个人健康哲学宣言 |
