# 芒格式投资学习计划：10次×2小时

> 基于芒格的工作方式和投资哲学
> 核心：阅读 + 思考 + 清单 + 等待

---

## 芒格的每日工作安排

芒格一天的典型节奏（根据传记和采访整理）：

| 时间 | 活动 |
|------|------|
| 6:00-7:00 | 起床、阅读早报 |
| 7:00-9:00 | 深度阅读（年报、书籍、财报） |
| 9:00-11:00 | 继续阅读+思考 |
| 11:00-12:00 | 步行/运动（用思考的时间） |
| 12:00-14:00 | 午餐+社交 |
| 14:00-16:00 | 阅读+分析公司 |
| 16:00-17:00 | 回复邮件、必要会议 |
| 17:00-18:00 | 继续阅读或者散步思考 |
| 晚上 | 阅读+早睡 |

**规律：每天4-6小时阅读，少量会议，大量独处思考。**

## 10次×2小时学习计划

### 第1次：建立投资认知框架

**阅读（45min）**
- 芒格关于能力圈的核心论述
- 理解能力圈 vs 能力边界

**练习（30min）**
- 写下你当前的投资能力圈：你真正懂的行业/公司是什么？
- 写下你明显不懂的领域
- 用「如果我问你是否超出了能力圈，你就已经超出了」这个标准检验

**思考（30min）**
- 你过去的亏损中，有多少是在能力圈外做的决策？
- 你的能力圈边界在哪里？怎么判断？

**复盘（15min）**
- 一个清晰的能力圈是什么样子的？
- 什么情况下你是在能力圈边界上？

**每日微任务**（每天10min）
- Day1-2：观察自己一天中在圈内和圈外做了哪些决策
- Day3-4：选择一家公司，判断它是否在你的能力圈内
- Day5-7：找一个人让他挑战你的能力圈判断

---

### 第2次：安全边际与好公司的标准

**阅读（45min）**
- 安全边际的概念和实际应用
- 好公司的三个标准：自由现金流、可持续竞争优势、优秀管理层

**练习（30min）**
- 选一家你关注的公司，用芒格的三个标准给它打分（1-10分）
- 它的自由现金流充足吗？护城河可持续吗？管理层值得信任吗？
- 现在的价格有安全边际吗？

**思考（30min）**
- 安全边际是数字还是原则？怎么判断「足够的安全边际」？
- 为什么说「以合理价格买好公司，好过以便宜价格买烂公司」？

**复盘（15min）**
- 好公司的三个标准中，你最看重哪一个？
- 你现在持有的投资符合这三个标准吗？

**每日微任务**（每天10min）
- Day1-3：研究一家好公司，找出它的三个标准
- Day4-5：找一家烂公司，分析它为什么不符合标准
- Day6-7：用安全边际分析一个你的持仓

---

### 第3次：护城河分析

**阅读（45min）**
- 五种护城河：品牌、规模优势、网络效应、专利/技术壁垒、特许经营权
- 可口可乐案例分析：多重护城河叠加

**练习（30min）**
- 选择一家你熟悉的公司，分析它的护城河属于哪种类型
- 它的护城河在变宽还是变窄？依据是什么？
- 如果竞争对手有无限资源，能不能抢走它的生意？

**思考（30min）**
- 没有护城河的生意长期会怎样？
- 护城河和短期增长哪个更重要？

**复盘（15min）**
- 你见过哪些公司护城河被侵蚀了？
- 你当前持有的投资有没有护城河？

**每日微任务**（每天10min）
- Day1-2：观察你生活中消费的品牌，想它们的护城河是什么
- Day3-4：找一家护城河在变窄的公司
- Day5-7：用护城河理论分析一只你关注的股票

---

### 第4次：检查清单设计

**阅读（45min）**
- 芒格的投资原则检查清单：风险、独立、准备、谦虚、严格分析、配置、耐心、决心
- 飞行员的检查清单设计逻辑

**练习（30min）**
- 根据芒格的框架，设计你自己的投资检查清单（至少10项）
- 样例：
  - [ ] 这家公司在我的能力圈内吗？
  - [ ] 它的护城河是什么？可持续吗？
  - [ ] 现在的价格有安全边际吗？
  - [ ] 如果我买入后跌30%，我能接受吗？
  - [ ] 三个最坏的情况是什么？
  - [ ] 我受到什么偏误的影响？
  - [ ] 有没有人能挑战我的判断？
  - [ ] 我准备好持有多久？（至少3年）
  - [ ] 这笔投资占我总资产的比例合理吗？
  - [ ] 如果明天停牌一年，我还愿意买吗？

**思考（30min）**
- 你过去的错误决策中，有没有哪些是走一遍清单就能避免的？
- 检查清单的核心作用是预防什么？

**复盘（15min）**
- 一份好的检查清单应该有多长？
- 你会在什么情况下使用它？

**每日微任务**（每天10min）
- Day1-2：起草你的第一版检查清单
- Day3-4：用清单回顾你过去的一个投资决策
- Day5-7：用小额投资走一遍清单，记录过程

---

### 第5次：反过来想与反面证据

**阅读（45min）**
- 反过来想的完整方法论
- 红队演练：主动寻找反面证据

**练习（30min）**
- 选一个你现有的投资，做「反过来想」：
  - 如果这笔投资一定会亏钱，三个可能的原因是什么？
  - 我怎么避免这些原因？
- 做一次红队演练：列出所有不支持这个投资的理由
- 如果这些理由成立，我的投资逻辑还站得住吗？

**思考（30min）**
- 反过来想是不是悲观？它和风险管理的区别是什么？
- 为什么在投资中找反面证据比找支持证据更重要？（确认偏误）

**复盘（15min）**
- 你最近一次找到反面证据并改变了判断，是什么时候？
- 反过来想改变了你什么决策？

**每日微任务**（每天10min）
- Day1-2：对一个日常决策做「反过来想」
- Day3-4：对一个投资决策列出3个反面证据
- Day5-7：找一个信任的人，让他挑战你的一个判断

---

### 第6次：情绪控制与性格修炼

**阅读（45min）**
- 做投资不需要高智商，需要能控制自己的情绪
- 双轨分析：理性轨+心理轨并行
- 误判心理学中的关键偏误：损失厌恶、社会认同、过度乐观、确认偏误

**练习（30min）**
- 回忆你最近的一个情绪化投资决策，发生了什么？
- 当时是哪个偏误在影响你？
- 如果走双轨分析，你会做什么不同？

**思考（30min）**
- 为什么聪明人做投资容易赔钱？
- 你的性格中，哪个偏误最容易导致你犯错？

**复盘（15min）**
- 双轨分析怎么帮你控制情绪？
- 你怎么在亏钱的时候保持理性？

**每日微任务**（每天10min）
- Day1-2：记录一天中的情绪变化对决策的影响
- Day3-4：练习一次双轨分析：先做理性分析，再做心理觉察
- Day5-7：创建一个你的偏误清单（你最常犯的5个）

---

### 第7次：Lollapalooza效应与多因素分析

**阅读（45min）**
- Lollapalooza效应：多因素同向叠加产生极端效果
- 在投资中如何识别和利用Lollapalooza

**练习（30min）**
- 找一家你关注的公司，列出影响它的所有因素：行业趋势、竞争格局、管理层、估值、市场情绪、政策环境
- 这些因素是同一方向还是相反方向？
- 如果多个负面因素同时出现，最坏的情况是什么？

**思考（30min）**
- 你经历过Lollapalooza效应吗？（正的或负的）
- 在投资中，怎么避免被负面的Lollapalooza影响？

**复盘（15min）**
- 多因素分析和单一因素分析的区别是什么？
- 你现在面临的最大投资决策可以列出几个因素？

**每日微任务**（每天10min）
- Day1-2：观察一个日常现象中的多因素叠加
- Day3-4：用多因素分析一个你关注的公司
- Day5-7：分析一次你的Lollapalooza经历

---

### 第8次：集中投资 vs 分散投资

**阅读（45min）**
- 分散投资是保护无知的一种方式
- 集中投资的逻辑和条件
- 一生只需做少数几个重大决策

**练习（30min）**
- 评估你当前的投资组合：持有了多少只股票/基金？
- 如果你的能力圈内只有3-5个机会，你敢集中吗？
- 用芒格的标准：你现在的组合是在保护无知还是在集中火力？

**思考（30min）**
- 集中投资的前提是什么？（能力圈+安全边际+长期）
- 如果只能选3只股票持有一辈子，你会选哪3只？

**复盘（15min）**
- 你现在的投资组合需要调整吗？怎么调整？
- 你距离芒格式的集中投资还有多远？

**每日微任务**（每天10min）
- Day1-2：列出你能力圈内最好的3个投资机会
- Day3-4：如果只能选一个，怎么选？
- Day5-7：设计一个集中投资组合（不超过5只）

---

### 第9次：公司深度分析实战

**阅读（45min）**
- 用芒格框架分析一家公司的完整流程
- 参考比亚迪案例：芒格怎么分析王传福

**练习（30min）**
- 选择一家你感兴趣的公司（最好是你能力圈内的）
- 走完完整分析：
  1. 它在我的能力圈内吗？
  2. 它的护城河是什么？可持续吗？
  3. 它的管理层值得信任吗？
  4. 它的自由现金流怎么样？
  5. 现在的价格有安全边际吗？
  6. 反面证据是什么？
  7. 我的检查清单走完了吗？
  8. 如果买入，我准备持有多久？

**思考（30min）**
- 这个分析流程和你以前的做法有什么不同？
- 你找到了什么以前没注意到的问题？

**复盘（15min）**
- 芒格的框架给你的分析增加了什么价值？
- 你做完这次分析，敢不敢重仓？

**每日微任务**（每天15min）
- Day1-7：每天花15分钟研究这家公司，一周后形成完整分析报告

---

### 第10次：建立你的投资系统

**阅读（45min）**
- 回顾芒格所有投资原则
- 从分散的点到系统：能力圈+安全边际+护城河+检查清单+集中投资+等待

**练习（30min）**
- 建立你的投资系统文档（写下来）：
  1. 你的能力圈描述（你懂什么、不懂什么）
  2. 你的检查清单
  3. 你的买入标准
  4. 你的卖出规则
  5. 你的组合管理原则
  6. 你的情绪控制流程
- 写下你从10次学习中学到的最重要的3个原则

**思考（30min）**
- 芒格的投资哲学中，哪一个原则对你改变最大？
- 你接下来打算怎么把这个系统应用到实际投资中？

**复盘（15min）**
- 你现在的投资系统还有什么漏洞？
- 你打算多久复盘和更新一次这个系统？

**后续每日惯例**

| 时间 | 活动 |
|------|------|
| 每天 | 阅读至少1小时（财报/书籍/新闻） |
| 每周 | 完成一次公司分析（可以不是完整的，但要走一遍框架） |
| 每月 | 复盘一次投资决策，对照检查清单 |
| 每季 | 回顾投资组合，检查是否在能力圈内 |
| 每年 | 更新投资系统文档，升级你的框架 |
