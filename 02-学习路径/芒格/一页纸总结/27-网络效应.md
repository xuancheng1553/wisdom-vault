# 一页纸总结：网络效应

## 一句话定义
> 一个产品的价值随着使用它的人数的增加而增加。用户越多，价值越大。

## 核心概念
1. **直接网络效应**：用户越多，每个用户的价值越大（如微信）
2. **间接网络效应**：用户越多，吸引的服务越多，价值越大（如苹果App Store）
3. **双边网络效应**：买方越多，吸引的卖方越多，反之亦然（如美团）
4. **冷启动问题**：初期用户太少，价值不够，需要先突破冷启动阈值
5. **赢家通吃**：网络效应强的行业容易出现一家独大的局面

## 真实案例
- 你的社群：付费用户越多，内容越好，内容越好，吸引更多人付费
- 微信：一个人用的时候没什么价值，所有人都在用你就不得不用
- 美团：商家越多，用户选择越多，用户越多，商家越愿意入驻
- 伯克希尔：芒格和巴菲特的声誉让好公司主动来找他们

## 常见错误
| 错误 | 表现 |
|------|------|
| 忽视冷启动 | 以为网络效应会自动发生 |
| 追求数量不追求质量 | 拉来一堆不活跃用户，网络效应也不强 |
| 过早收费 | 还没到临界点就收费，用户增长停滞 |
| 忽视竞品网络 | 对手的网络比你大，用户很难被吸引过来 |
