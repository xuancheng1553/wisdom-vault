# 一页纸总结：均值回归

## 一句话定义
> 极端情况不会永远持续，一切终将回到平均水平。涨多了会跌，跌多了会涨。

## 核心概念
1. **回归本质**：无论是公司业绩还是个人表现，极端好或坏之后都会向均值回归
2. **警惕追涨**：连续大涨后大概率会回落，追涨是最危险的策略
3. **不要恐慌抛售**：连续大跌后大概率会反弹，恐慌卖出是最亏的策略
4. **长期看是均值**：短期波动是正常的，长期会回归均值
5. **均值不是静止的**：均值本身也会随着基本面变化。一家好公司的均值可能在上升
6. **识别真假回归**：区分正常波动和趋势改变是关键

## 真实案例
- 股市：连续涨了3年的行业，未来2年大概率跑输大盘
- 你的资产配置：某个资产大涨后，再平衡就是在利用均值回归
- 个人表现：上个月业绩特别好，这个月回落了——不是说你不努力了，是回归均值

## 常见错误
| 错误 | 表现 |
|------|------|
| 把均值回归当趋势 | 以为涨多了的还会继续涨 |
| 在极端点追涨 | 在别人最狂热时买入 |
| 在极端点恐慌卖出 | 在别人最恐慌时卖出 |
| 忽略均值变化 | 公司的基本面变了，均值已经不是原来的均值了 |

## 检查清单
- [ ] 当前是极端情况还是正常范围？
- [ ] 我是在追涨还是恐慌抛售？
- [ ] 基本面有改变吗？还是只是价格波动？
- [ ] 我现在的仓位是在均值上还是远离均值？
