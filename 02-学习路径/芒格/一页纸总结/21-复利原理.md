# 一页纸总结：复利原理

## 一句话定义
> 复利是世界第八大奇迹。每天进步一点点，长期积累的效果惊人。F = P(1+r)^n，本金、收益率、时间是三大核心变量。

## 核心概念
1. **指数增长的本质**：复利不是线性增长，是指数增长。初期增长缓慢，越过临界点后加速
2. **时间是最关键变量**：时间越长，复利效果越惊人。30年的复利远比20年的复利高出数倍
3. **收益率差异的影响**：年化10%和15%，30年后差距接近4倍
4. **本金的重要性**：本金越大，复利效果越显著。这就是为什么年轻时要节俭积累
5. **复利不仅适用于金钱**：知识积累、健康锻炼、人际关系都适用复利规律
6. **中断是复利的敌人**：频繁中断会破坏复利效应

## 真实案例
- 芒格每天阅读4-6小时，坚持50年，从一个普通律师变成最聪明的投资者之一
- 你从99元群开始，到3600元群，每一步都在积累认知和信誉的复利
- 每天学习30分钟投资知识，一年后你就超越了90%的投资者
- 每天存100元，年化10%，30年后你将拥有约200万

## 常见错误
| 错误 | 表现 |
|------|------|
| 低估时间 | 投了3个月就想看到效果 |
| 中断复利 | 频繁交易、频繁换方向 |
| 追求暴利 | 高收益必然伴随高风险，破坏复利基础 |
| 忽略本金积累 | 花太多时间研究高收益，花太少时间增加本金 |
