# 一页纸总结：概率论与贝叶斯思维

## 一句话定义
> 不要用二元思维（会涨/不会涨）思考问题，用概率思维（70%概率会涨）代替。并根据新信息不断更新你的判断。

## 核心概念
1. **概率不是0或1**：大多数事情是概率性的，不是确定性的。用百分比思考
2. **赔率思维**：不仅要看赢的概率，还要看赢多少、输多少。期望值 = 胜率 x 赢的金额 - 败率 x 输的金额
3. **贝叶斯更新**：先有一个初始判断（先验概率），然后每获得一条新信息就更新一次判断（后验概率）
4. **不要被小概率事件吓到**：关注期望值，而不是恐惧极端可能性
5. **决策质量≠结果质量**：一个好的决策可能带来坏的结果，反之亦然。长期按概率决策你就会赢

## 真实案例
- 投资决策：不要说「这只股票肯定涨」，要说「我有70%的把握会涨，但要做好30%不涨的准备」
- 你的培训社群：不是「一定有人付费」，而是「有60%的概率有人付费，所以我先试99元」
- 贝叶斯更新：先认为一家公司有价值（初始判断），读一份财报后调整判断，再读一份再调整

## 常见错误
| 错误 | 表现 |
|------|------|
| 二元思维 | 要么肯定涨要么肯定跌 |
| 忽视赔率 | 只看概率不看赔率 |
| 不更新判断 | 有了新信息也不调整原有判断 |
| 被小概率事件吓到 | 过度关注罕见风险 |

## 检查清单
- [ ] 我对这个判断的概率是多少？为什么？
- [ ] 如果错了，最坏的结果是什么？我能承受吗？
- [ ] 我有没有根据最新信息更新判断？
- [ ] 期望值算过了吗？（不是只算概率）
