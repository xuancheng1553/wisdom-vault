# 一页纸总结：公地悲剧

## 一句话定义
> 共享资源会被人过度使用直至耗尽。每个人只考虑自己的利益，不考虑对资源的长期伤害。

## 核心概念
1. **个人利益vs集体利益**：每个人多占一点资源对自己有利，但大家多占了资源就完了
2. **产权不清晰是根本原因**：没人拥有就没有人保护
3. **搭便车问题**：我不维护也会有别人维护，所以我就不维护了
4. **芒格的解法**：明确产权、建立制度、让使用者成为所有者
5. **公共资源管理的条件**：边界清晰、共同商定规则、对违规有惩罚

## 真实案例
- 微信群氛围变差：每个人都发广告，结果群就废了（需要群规）
- 公共渔场：每个人都多捕一点，最终鱼被捕光了
- 你的社群：如果每个人都在群里发广告，社群就失去价值了

## 常见错误
| 错误 | 表现 |
|------|------|
| 无管理共享 | 没有规则也没有管理者 |
| 问题出现后介入太晚 | 资源已经枯竭了才采取措施 |
| 惩罚不够 | 违规成本太低 |

## 检查清单
- [ ] 我管理的资源有明确的规则吗？
- [ ] 违规行为有足够的惩罚吗？
- [ ] 用户/成员对资源的长期健康有责任感吗？
