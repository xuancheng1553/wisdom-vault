# 一页纸总结：熵增定律

## 一句话定义
> 封闭系统会自然趋向混乱和无序。必须持续输入能量才能维持秩序。

## 核心概念
1. **熵的定义**：熵是系统混乱程度的度量。熵越高越混乱
2. **封闭系统必死**：不与外界交换能量的系统一定会走向混乱和衰亡
3. **对抗熵增需要能量**：为了维持秩序，你必须持续输入能量（时间、金钱、努力）
4. **开放系统才能持久**：持续吸收新信息、新资源才能对抗熵增
5. **应用到生活**：不学习就会变笨、不运动就会变胖、不维护关系就会疏远

## 真实案例
- 你的社群：如果不持续输入新内容，用户会流失，社群会死掉
- 投资组合：不持续研究和调整，持仓会偏离你的配置目标
- 个人成长：不持续学习，认知会退化
- 房间不收拾：一周不收拾就乱了——混乱是默认状态，秩序需要持续维护

## 常见错误
| 错误 | 表现 |
|------|------|
| 以为秩序会自动维持 | 觉得做好了就可以不管了 |
| 封闭系统运行 | 不吸收新信息、新知识 |
| 忽视早期信号 | 等到混乱已经发生了才处理 |

## 检查清单
- [ ] 我在为我的系统持续输入能量吗？
- [ ] 我的知识/业务在有序化还是走向混乱？
- [ ] 我有没有在对抗我的自然退化？
- [ ] 我的系统是开放还是封闭的？
