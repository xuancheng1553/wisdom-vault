# 一页纸总结：芒格投资哲学

## 一句话定义
> 在能力圈内、以安全边际买入好公司、集中持有、长期等待。

## 核心概念
1. **能力圈**：只投你真正懂的行业，不懂的不碰
2. **安全边际**：以低于内在价值的价格买入，留足安全垫
3. **好公司的标准**：自由现金流+护城河+优秀管理层
4. **集中投资**：找到好机会就下重注，不是分散一百个
5. **耐心等待**：大部分时间什么都不做，等待最好的机会出现
6. **情绪控制**：性格比智商重要
7. **检查清单**：重大决策前走一遍

## 真实案例
- 比亚迪：芒格看好王传福这个人，重仓买入，持有多年获得巨大回报
- 伯克希尔：一生中最重要的10个投资决策占了总收益的90%以上

## 常见错误
| 错误 | 表现 |
|------|------|
| 在能力圈外投资 | 听消息买不懂的股票 |
| 没有安全边际 | 追高买入，回调就套牢 |
| 过度分散 | 买了几十只股票，不如买ETF |
| 频繁交易 | 把投资当赌博 |
| 情绪化决策 | 涨了兴奋跌了恐慌 |

## 检查清单
- [ ] 这家公司在我的能力圈内吗？
- [ ] 我以什么价格买入？有安全边际吗？
- [ ] 它的护城河是什么？能持续多久？
- [ ] 如果我买入后跌了50%，我能接受吗？
- [ ] 我打算持有多久？（至少3-5年）
- [ ] 我走完检查清单了吗？
