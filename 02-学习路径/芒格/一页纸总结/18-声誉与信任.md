# 一页纸总结：声誉与信任

## 一句话定义
> 你的声誉是你最宝贵的资产，可能需要20年才能建立，但5分钟就能毁掉。

## 核心概念
1. **声誉即资产**：比金钱更重要的资产
2. **长期积累**：需要20年建立，5分钟毁掉
3. **信任是社交货币**：没有信任就没有合作
4. **一致性**：说到做到是建立信任的最佳方式

## 真实案例
- 芒格一生不做任何可能损害声誉的事
- 伯克希尔收购案例中，他坚持对卖方的承诺，即使合同没有写明

## 常见错误
| 错误 | 表现 |
|------|------|
| 以为声誉不重要 | 做了损害信誉的事 |
| 言行不一致 | 说一套做一套 |
| 短期行为 | 为了短期利益损害长期信誉

## 检查清单
- [ ] 我最近的行为有没有损害我的声誉？
- [ ] 我说到做到了吗？
- [ ] 别人信任我吗？
