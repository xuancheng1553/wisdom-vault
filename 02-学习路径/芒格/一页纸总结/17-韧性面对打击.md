# 一页纸总结：韧性面对打击

## 一句话定义
> 生活一定会给你打击，关键是你能不能继续前进。

## 核心概念
1. **接受打击**：芒格30岁失明、中年丧子、经历离婚
2. **韧性不是天生的**：是在一次次打击中练出来的
3. **继续前进**：关键不是打击多大，而是你停了没有
4. **从打击中学习**：每次打击都带来一次认知升级

## 真实案例
- 芒格30岁因白内障手术失败失去一只眼睛，学会用一只眼睛阅读
- 9岁的儿子因白血病去世，他后来把精力投入到慈善
- 经历离婚后重新找到幸福婚姻，持续55年

## 常见错误
| 错误 | 表现 |
|------|------|
| 被打倒就躺平 | 一次失败就放弃 |
| 拒绝承认打击 | 假装没事不处理情绪 |
| 抱怨环境 | 把责任推给别人不行动
