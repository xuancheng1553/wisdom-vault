# 一页纸总结：帕金森定律

## 一句话定义
> 工作会膨胀到占满所有可用的时间。给你多少时间，你就会花多少时间。

## 核心概念
1. **时间膨胀**：给你2小时能做完的事，给8小时就会花8小时
2. **效率来自限制**：时间越紧迫，效率越高
3. **完美主义的根源**：没有截止日期就没有完成
4. **应用：给自己设限**：给任务设置比实际需要更短的时间
5. **芒格的做法**：每天安排大量阅读，是因为他知道如果不安排时间，就会被会议和琐事填满

## 真实案例
- 你的一份报告：如果截止日期是周五，你周四开始写，周五完成。如果截止日期是下个月，你会一直拖到下个月
- 芒格的日程：他主动安排4-6小时阅读，不然时间会被会议吞掉
- 你的社群内容：给自己设一个「每周一发布」的死线，保证产出

## 常见错误
| 错误 | 表现 |
|------|------|
| 不给任务设时间 | 没有截止日期就没有完成 |
| 低估时间膨胀 | 以为时间充裕就能做得更好 |
| 多个任务互相抢占 | 没有优先级，时间被填满但产出很低 |

## 检查清单
- [ ] 我有没有给任务设定明确的截止时间？
- [ ] 这个任务如果时间减半能完成吗？
- [ ] 我的时间是被主动安排的还是被动填满的？
