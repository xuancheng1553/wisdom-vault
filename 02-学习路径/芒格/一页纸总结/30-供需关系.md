# 一页纸总结：供需关系

## 一句话定义
> 价格由供给和需求决定。供过于求价格下跌，供不应求价格上涨，偏离一定会回归。

## 核心概念
1. **价格信号**：价格是最直接的供需信号。价格上涨说明需求大于供给
2. **供给弹性**：有些商品供给可以快速增加（如服装），有些不行（如稀缺资源）
3. **需求弹性**：有些商品价格上涨后需求大幅下降（如奢侈品），有些不会（如食盐）
4. **均衡价格**：供给量和需求量相等时的价格
5. **外部因素影响**：政策、技术、社会变化都会改变供需曲线
6. **长期与短期**：短期供需失衡常见，长期会趋于均衡

## 真实案例
- 你收费从99元到3600元 → 价格上升但是用户数下降（需求弹性）
- 疫情时口罩价格暴涨 → 短期需求暴增，供给跟不上
- 油价下跌 → 供过于求，经济放缓需求下降
- 优质资产配置顾问稀缺 → 供给有限但需求大，所以你有机会

## 常见错误
| 错误 | 表现 |
|------|------|
| 只看供给不看需求 | 做自己想做但没人需要的事 |
| 忽视供给侧竞争 | 觉得市场大就做，没看到竞争对手 |
| 定价不看供需 | 凭感觉定价 |
| 短期供需长期化 | 把短期供过于求当成长期趋势 |

## 检查清单
- [ ] 我的产品/服务是供过于求还是供不应求？
- [ ] 我的定价基于供需关系还是成本？
- [ ] 供给侧的竞争格局是什么？
- [ ] 需求是短期现象还是长期趋势？
