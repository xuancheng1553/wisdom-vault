# 一页纸总结：比较优势

## 一句话定义
> 每个人/每个国家应该做自己最擅长的事，通过交换获得更多。

## 核心概念
1. **做你相对最好的**：不是做你最好的，是做你相对别的事情最好的
2. **通过交换获益**：你做你擅长的，我做我擅长的，交换后大家都得到更多
3. **专注与合作的逻辑**：专注能让你更擅长，合作能弥补你的不足
4. **能力圈的应用**：在能力圈内深耕，圈外的事找合作
5. **动态变化**：你的比较优势不是固定的，可以培养

## 真实案例
- 你做了资产配置群而不是自己去炒股——你在资产配置上有比较优势
- 芒格做投资而不是继续做律师——投资是他的比较优势
- 伯克希尔投比亚迪而不是自己造车——有投资优势没有造车优势
- 一个人做所有事（内容、运营、客服、技术）不如找到合作伙伴

## 常见错误
| 错误 | 表现 |
|------|------|
| 什么都自己做 | 没有利用交换和合作 |
| 在劣势领域硬扛 | 在自己不擅长的领域费很大劲 |
| 忽视培养优势 | 只在已有的优势上吃老本不提升 |

## 检查清单
- [ ] 我的比较优势是什么？
- [ ] 我有没有在非优势领域浪费精力？
- [ ] 我可以和谁合作来弥补我的不足？
- [ ] 我怎么提升我的比较优势？
