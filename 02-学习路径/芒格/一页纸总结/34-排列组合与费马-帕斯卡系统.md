# 一页纸总结：排列组合与费马-帕斯卡系统

## 一句话定义
> 用数学方法计算不同选择的概率和赔率，做出最优决策。

## 核心概念
1. **计算可能性的数量**：排列组合帮你理解「有多少种可能的结果」
2. **费马-帕斯卡系统**：用数学计算赔率，而不是凭感觉判断
3. **决策树**：列出所有可能的分支，为每个分支赋予概率和结果值
4. **期望值计算**：每个选项的期望值 = 概率 x 收益之和
5. **长期优势**：不追求每次都对，追求长期做正期望值的决策

## 真实案例
- 投资决策：这只股票有60%概率涨20%，40%概率跌10%，期望值 = 60%x20% - 40%x10% = 8%，正期望值可以投
- 创业决策：做3600元群，有50%概率成功（赚20万），50%概率失败（亏2万），期望值 = 50%x20万 - 50%x2万 = 9万

## 常见错误
| 错误 | 表现 |
|------|------|
| 不计算概率 | 凭感觉决策 |
| 忽视极端事件 | 没有考虑小概率但影响巨大的事件 |
| 公式误用 | 计算了但用了错误的概率和收益值 |
| 过度依赖模型 | 忘了模型只是对现实的简化 |
