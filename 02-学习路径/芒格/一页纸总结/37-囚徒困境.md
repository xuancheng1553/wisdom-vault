# 一页纸总结：囚徒困境

## 一句话定义
> 个体理性选择会导致集体非理性结果。每个人都做对自己最有利的事，结果所有人都更差。

## 核心概念
1. **个人理性≠集体理性**：每个人做对自己最有利的事，合起来可能是最差的
2. **信任是关键**：囚徒困境的核心是不信任——如果你信对方也会合作，合作是最好的
3. **重复博弈改变结果**：如果知道会长期合作，背叛的代价就变大了
4. **声誉的作用**：声誉让你在还不知道结果的情况下也愿意合作
5. **芒格的解法**：建立长期合作关系、选择值得信任的人、建立声誉

## 真实案例
- 行业价格战：一家降价→其他跟着降→所有人都赚更少
- 你帮了别人，别人不帮你：短期看别人占了便宜，长期看没人再帮TA
- 芒格和巴菲特60年的合作：长期重复博弈让合作成为最优选择

## 常见错误
| 错误 | 表现 |
|------|------|
| 短期思维 | 只看一次博弈，不考虑长期后果 |
| 过度警惕 | 以为所有人都在背叛，结果自己也背叛 |
| 忽视声誉 | 以为一次背叛没什么，不知道声誉成本有多大 |

## 检查清单
- [ ] 我是在和对方长期合作还是短期博弈？
- [ ] 我的决定对对方有什么影响？
- [ ] 我的声誉在这场合作中重要吗？
- [ ] 有没有更好的方式让双方都受益？
