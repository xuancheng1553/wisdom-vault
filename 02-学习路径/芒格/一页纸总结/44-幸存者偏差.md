# 一页纸总结：幸存者偏差

## 一句话定义
> 我们只看到成功者，忽略了失败者，从而高估了成功的概率。

## 核心概念
1. **视线偏差**：成功者被我们看到，失败者消失了——统计上这叫幸存者偏差
2. **抛硬币错觉**：如果有一万个投资者抛硬币，总有几个能连续抛中10次——但这不是因为他们技术好
3. **成功学的陷阱**：成功者归因于自己的方法，但同样方法做的人可能99%都失败了
4. **芒格的逆向思维**：研究失败比研究成功更有价值——去研究那些死了的公司
5. **正确的方法**：不只研究成功案例，也研究失败案例

## 真实案例
- 你只看到做投资社群成功的人，没看到100个做同样事失败的人
- 你看到巴菲特成功了，但用同样方法的人99%没有他的结果
- 芒格的方法是：建立失败清单——研究那些死了的公司

## 常见错误
| 错误 | 表现 |
|------|------|
| 只看成功者 | 只学习成功案例，不研究失败 |
| 归因错误 | 把成功归因于方法而不是概率 |
| 忽视样本偏差 | 以为看到的样本就代表全部 |
