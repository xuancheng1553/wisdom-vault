# 一页纸总结：激励机制

## 一句话定义
> 激励机制是世界上最强大的力量。你不需要讲道理，只需要设计好激励。

## 核心概念
1. **激励的力量**：想让人们做什么就给他们相应的激励，想让人们不做什么就让他们承担后果
2. **激励错位**：很多制度的失败是因为激励错了方向
3. **识别激励**：理解任何行为背后的激励因素
4. **自我激励**：对自己也要设计合理的激励机制

## 真实案例
- 芒格说联邦快递的夜间分拣系统之所以高效，是因为夜班员工按件计薪
- 你的培训社群：为什么有人愿意付99元？因为内容对他有用（激励对了方向）

## 常见错误
| 错误 | 表现 |
|------|------|
| 激励错方向 | 想要A却奖励B |
| 忽视负面激励 | 让好事不做让坏事做 |
| 只用金钱激励 | 忽视了认可/成就感/自主权

## 检查清单
- [ ] 我当前的行为受什么激励驱动？
- [ ] 我设计的激励在鼓励正确的行为吗？
- [ ] 我对自己的激励有效吗？
