# 一级市场投资学习路径：5级进阶体系

## 设计理念
一级市场投资（VC/PE/天使投资）与二级市场有本质区别：没有公开价格、流动性差、信息不对称严重、失败率极高。这套路径从"看懂项目"到"看懂行业"再到"看懂人性"，对标顶级投资机构的分析师成长路径。

---

## Level 1：认知入门
> 目标：理解一级市场是什么、生态链如何运作、和二级市场的本质区别

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 核心认知 | 一级市场是"把钱投到未上市公司的股权中"——没有K线、没有交易量、没有实时报价 | 能用一句话说清一级市场和二级市场的区别 |
| 生态地图 | LP→GP→被投企业→投后管理→退出（IPO/并购/回购） | 能画出完整的资金流转图 |
| 角色分工 | 天使投资、VC（种子/早期/成长期）、PE（成长/并购）、战略投资 | 能说出各阶段的特点和代表机构 |
| 赚钱逻辑 | 管理费（2%）+ Carry（20%）——但真正赚钱的是Carry | 理解GP的商业模式 |
| 常见错误 | 把二级市场的思维带进一级（看K线、追热点）、低估失败率（90%的早期项目会死） | 避免两大入门误区 |
| 阅读起点 | 《风险投资史》前3章、《硅谷来信》关于VC的文章 | 建立认知地图 |
| 进阶标准 | 能用大白话讲清楚"一级市场是干什么的"，知道VC和PE的区别 | 完成阅读任务 |

## Level 2：方法掌握
> 目标：掌握项目筛选和基础分析的框架

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 行业研究 | PEST分析、波特五力、产业链图谱、市场规模测算（TAM/SAM/SOM） | 能独立完成一个行业的初步分析 |
| 商业模式分析 | 价值主张、收入模型、成本结构、单位经济模型（LTV/CAC）、护城河 | 能说清一个项目的"钱是怎么赚的" |
| 团队评估 | 创始人背景、团队完整性、股权结构、激励机制、化学反应 | 能判断一个团队是否"靠谱" |
| 产品与技术 | MVP验证、技术壁垒、产品迭代能力 | 能判断产品的"最小可验证单元" |
| 财务基础 | 三张表解读、收入预测、现金流分析、烧钱率（Burn Rate） | 能看懂BP中的财务假设是否合理 |
| 常见错误 | 只看赛道不看团队、只听故事不看数据、把PPT当事实、过度关注估值忽视基本面 | 避免五大判断陷阱 |
| 进阶标准 | 能在30分钟内对一个项目做出初步的"好/不好"判断，并说出3个核心理由 | 独立完成3个项目的初步分析 |

## Level 3：实践应用
> 目标：掌握尽职调查、估值谈判和投资决策的全流程

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 尽职调查 | 业务DD、财务DD、法务DD、技术DD、团队DD——每类的核心问题清单 | 能写出一个完整的DD Checklist |
| 估值方法 | 可比公司法、DCF、风险因素汇总法（VC Method）、实物期权、估值区间思维 | 能对一个早期项目给出估值区间和理由 |
| 交易条款 | Term Sheet核心条款：清算优先权、反稀释、防稀释、董事会席位、对赌/回购 | 能读懂TS中每一条的实际含义 |
| 投资备忘录 | 投资逻辑、风险识别、退出路径、投后计划 | 能写出一份结构完整的投资备忘录 |
| 投资决策 | 决策树、反向清单（什么情况下不投）、异议机制 | 能建立自己的投资决策框架 |
| 常见错误 | 不做反调（只找支持投资的证据）、尽调浮于表面（不去一线访谈）、被创始人PUA、跟风投资 | 避免四大决策陷阱 |
| 进阶标准 | 独立完成一次完整的投决流程（从项目源到投决会），写出一份可经得起挑战的投资备忘录 | 完成1个项目的完整DD+TS+投决 |

## Level 4：系统思维
> 目标：建立组合管理、投后增值、退出策略的全局视角

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 组合管理 | 投资节奏、行业分布、阶段分布、风险对冲、跟投策略 | 能设计一个10-20个项目的投资组合 |
| 投后管理 | 战略支持、招聘对接、下一轮融资协助、董事会参与、危机处理 | 能制定一份投后服务清单 |
| 退出策略 | IPO条件、并购逻辑、S基金转让、回购执行、流动性管理 | 能对不同阶段项目制定退出预案 |
| 行业网络 | 创业者圈层、同行LP、FA/投行、专家网络、政府关系 | 能规划自己的人脉地图 |
| 基金运营 | 基金设立、LP关系管理、IR、Carry分配机制 | 理解基金的"生意逻辑" |
| 常见错误 | 投后不管、过度干预、退出犹豫（等更大的估值结果错过窗口）、组合集中度过高 | 避免四大管理陷阱 |
| 进阶标准 | 能管理一个5-10个项目的portfolio，并对每个项目有清晰的"下一步行动" | 完成一次portfolio review |

## Level 5：融会贯通
> 目标：形成自己的投资哲学，与芒格/巴菲特/纳瓦尔的智慧交叉印证

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 投资哲学 | 你投什么、不投什么、为什么——形成自己的"投资原则声明" | 写出一份完整的投资哲学说明书 |
| 一级×二级 | 二级市场的估值锚定效应、流动性溢价、一二级价差套利 | 能用二级市场的视角反视一级定价 |
| 芒格交叉 | 能力圈→只投你懂的行业、检查清单→尽调清单、反过来想→反向尽调 | 把芒格框架嵌入投资全流程 |
| 巴菲交叉 | 护城河→真正的壁垒是什么、安全边际→估值不能过于乐观、长期持有→陪伴企业成长 | 用巴菲特思想约束投资行为 |
| 纳瓦尔交叉 | 判断力>努力、只投方向和团队都对的、用专长做差异化投资 | 用纳瓦尔框架提升投资决策质量 |
| 常见错误 | 没有自己的投资哲学（跟风投资）、把方法论当教条（环境变了还在用旧框架）、失去好奇心（只看熟悉的赛道） | 避免三大融通陷阱 |
| 进阶标准 | 形成了自己的投资哲学和方法论体系，能在不同的市场环境中灵活调整 | 建立个人投资体系文档 |

## 关键数据基准

| 指标 | 种子轮 | A轮 | B轮 | C轮+ |
|------|--------|-----|-----|------|
| 典型估值 | 500-2000万 | 2000万-1亿 | 1-5亿 | 5亿+ |
| 典型投资额 | 50-500万 | 200-2000万 | 1000万-1亿 | 5000万+ |
| 成功率 | 1-5% | 3-8% | 10-20% | 20-30% |
| 典型持有期 | 7-10年 | 5-8年 | 3-5年 | 1-3年 |
| 决策周期 | 1-4周 | 1-3月 | 2-6月 | 3-12月 |

> 注：以上为中美市场大致参考，不同赛道差异很大。硬科技赛道估值系统性低于互联网，但持有期更长。
> 一级市场投资最重要的是"学会判断人的能力"——这是所有模型和数据都无法替代的。
> 芒格说"我宁愿和一只聪明的猴子谈一笔糟糕的交易，也不愿和一隻笨猴谈一笔好交易"——团队永远排第一。
