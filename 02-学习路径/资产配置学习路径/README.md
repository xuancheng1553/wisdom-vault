# 资产配置技能学习路径：5级进阶体系

## 设计理念
资产配置是"投资中唯一的免费午餐"——它对投资收益的贡献超过90%，是普通投资者最应该掌握、也最容易被忽视的能力。这套路径从"认识资产"到"建立系统"，让学员一步步学会自己管好自己的钱。

---

## Level 1：认知入门
> 目标：理解什么是资产配置、为什么它比选股更重要、核心思想地图

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 核心认知 | 资产配置是"把钱分配到不同种类的资产中"——不是靠选一只好股票发财，而是靠组合的稳定性盈利 | 能用一句话说清资产配置和选股的本质区别 |
| 思想地图 | 股债金/大类资产、风险收益特征、相关性、分散化、再平衡、长期主义 | 能说出6个核心概念 |
| 数据震撼 | 投资收益的90%+来自资产配置（Brinson研究）、择时贡献<5% | 理解为什么配置比选股重要 |
| 常见错误 | 以为资产配置=分散（买了10个股票就是分散）、以为配置足够就可以不管了 | 避免两大入门误区 |
| 进阶标准 | 对"不把鸡蛋放在一个篮子里"有了真正的理解而不是口头禅 | 能说出自己的资产应该怎么分 |

## Level 2：方法掌握
> 目标：掌握主要资产类别的特征和基本的组合构建方法

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 主要资产 | 股票（A股/港股/美股）、债券（国债/信用债）、黄金、现金、REITs | 能说出各类资产的年化收益和最大回撤区间 |
| 风险收益 | 收益≠风险免费午餐——高收益一定高风险，高风险不一定高收益 | 能画出各类资产的收益-风险散点图 |
| 相关性 | 资产之间的"亲疏关系"——负相关最好、正相关最差、低相关最理想 | 能说出股债之间的相关性和意义 |
| 组合构建 | 股债金三分法、60/40经典组合、目标日期法 | 能自己设计一个简单的股债金组合 |
| 常见错误 | 只看收益不看风险、选了高度相关的资产以为分散了 | 理解"真分散"和"假分散" |
| 进阶标准 | 能自己设计一个包含3-4类资产的简单组合 | 完成第一个组合设计 |

## Level 3：实践应用
> 目标：掌握再平衡和动态调整的方法，学会管理自己的投资组合

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 再平衡 | 定期间隔（半年/一年）或阈值触发（偏离5%）——低卖高买的制度化纪律 | 能制定自己的再平衡规则并执行 |
| 定投策略 | 定投降低择时风险，但不能改变资产本身的特征 | 对一个资产是否该定投能做出判断 |
| 仓位管理 | 满仓和空仓都是极端——建仓/加仓/减仓的节奏 | 能设计一个合理的建仓计划 |
| 风险控制 | 最大回撤管理、止损纪律、压力测试 | 能在亏损时保持理性而不是panic sell |
| 投后管理 | 定期复盘、调整比例、承认错误 | 能每季度做一次portfolio review |
| 常见错误 | 牛市全仓冲到最大风险、熊市最低点割肉、三天不赚钱就换方案 | 避免三大行为陷阱 |
| 进阶标准 | 独立管理自己的组合超过1年，经历过下跌没panic | 完成一次完整的市场周期管理 |

## Level 4：系统思维
> 目标：建立动态配置的能力，理解宏观因素对资产价格的影响

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 宏观框架 | 经济周期（复苏/过热/衰退/萧条）与资产的轮动规律 | 能说出不同周期阶段什么资产表现好 |
| 估值驱动 | 股债性价比（ERP）、估值温度计、历史百分位 | 能用估值指标辅助配置调整 |
| 因子视角 | 价值/成长/质量/低波——从大类资产到因子配置 | 理解因子分散比资产分散更深一层 |
| 动态调整 | 战略配置（SAA）和战术配置（TAA）的区别和配合 | 能同时运用两层框架 |
| 极端预案 | 黑天鹅、极端行情的应对策略 | 能为自己的组合做压力测试 |
| 常见错误 | 频繁切换战略、过度自信以为能预测宏观 | 区分SAA和TAA的边界 |
| 进阶标准 | 能在不同市场环境中保持策略一致性 | 建立自己的资产配置系统文档 |

## Level 5：融会贯通
> 目标：将资产配置智慧与其他智慧体系交叉印证

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| ×芒格 | 能力圈（不投不懂的资产）、逆向投资（极度低估时加仓）、安全边际（配置时留足现金） | 写一篇对比分析 |
| ×毛主席 | 实事求是（配置方案要根据自己的情况）、矛盾论（主要风险和次要风险）、实践论（从简单开始迭代） | 写一篇对比分析 |
| ×纳瓦尔 | 判断力优先（配置方向比选哪个ETF重要）、长期主义（越简单的配置越能坚持） | 写一篇对比分析 |
| ×黄帝内经 | 整体观（组合和身体一样是系统）、阴阳平衡（股债平衡）、因时制宜（经济周期如四季） | 写一篇对比分析 |
| ×稻盛和夫 | 持续力量（再平衡就是持续）、付出不亚于任何人的努力（研究配置框架需要投入） | 写一篇对比分析 |
