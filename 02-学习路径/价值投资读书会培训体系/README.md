# 价值投资读书会培训体系——基于你所有资料的综合课程

## 体系总览
基于你过去积累的所有材料——从《小狗钱钱》入门到深度公司分析、从资产配置到交易规则、从读书会运营到内容营销——这套培训体系把你的"碎片化材料"串成一条完整的学习路径。

---

## Module 1：投资认知入门——从0到1的思维转变

> 核心材料：小狗钱钱、聪明的投资者、有钱人想的和你不一样、穷人思维vs富人思维

| 维度 | 内容 | 对应资料 |
|------|------|----------|
| 什么是钱 | 钱是工具不是目的——小狗钱钱的核心：养一只会下金蛋的"鹅" | 《小狗钱钱》全书、《有钱人想的和你不一样》 |
| 投资vs投机 | 格雷厄姆定义了"投资"和"投机"的根本区别 | 《聪明的投资者》第一章 |
| 富人思维 | 穷人关注存量（我有多少钱），富人关注增量（我能创造什么价值） | "02比贫穷更可怕的 是穷人思维.doc" |
| 复利的魔力 | 时间是你唯一的杠杆——10%年化vs20%年化差30倍 | 聪明的投资者_投资策略 |
| 常见错误 | 把"理财"当"发财"、一夜暴富心态、不学习就入场 | 读书会学员常见问题 |

**配套阅读**：《小狗钱钱》→《聪明的投资者》第一章→《有钱人想的和你不一样》

## Module 2：资产配置体系——管好钱的核心技能

> 核心材料：三种配置策略对比回测、股债平衡投资方法论、资产配置资金表、温度计算、QDII基金列表、指数基金估值追踪

| 维度 | 内容 | 对应资料 |
|------|------|----------|
| 三策略对比 | 纯股策略 vs 股债平衡 vs 股债金——不同配置的中长期表现差异 | 《三种配置策略对比回测报告》 |
| 股债平衡 | 股债平衡投资方法论：比例设定、再平衡规则、实操步骤 | 《股债平衡投资方法论-v1.0》、吴晓波资产配置课 |
| 资产配置实操 | 用资金表管理你的配置——每月更新、每季调整 | 《资产配置资金表（空白）.xls》、《客户配置表案例》 |
| QDII与全球配置 | 为什么要配置海外——美股、港股、新兴市场 | 《QDII基金列表.xlsx》 |
| 温度计算 | 用估值温度判断市场冷暖——低温加仓、高温减仓 | 《温度计算.xlsx》、《指数基金估值追踪表》 |

**配套工具**：资产配置资金表模板、温度计算器、估值追踪表

## Module 3：公司基本面分析——看懂一家好公司

> 核心材料：公司基本面分析框架表 4.3、九模块分析框架、双框架分析

| 维度 | 内容 | 对应资料 |
|------|------|----------|
| 九模块框架 | 行业-公司-成长-盈利-估值-风险-管理层-财务-催化——9个维度完整覆盖 | 《公司基本面分析框架表 4.3》、锐明技术/时代电气深度报告 |
| 财务分析基础 | 看三张表——利润表（赚多少）、资产负债表（有多少）、现金流量表（现金有多少） | 圣邦股份年报、各公司深度报告 |
| 护城河判断 | 技术壁垒vs品牌vs规模vs网络效应——好公司一定有一条护城河 | 各深度报告中的"竞争优势"部分 |
| 双框架分析 | 将基本面分析（值不值得买）和交易规则（什么时候买、什么时候卖）结合 | 《力量钻石_双框架分析》、《奕瑞科技_双框架分析》 |

**配套阅读**：先读"公司基本面分析框架表 4.3"→再读2-3篇深度报告→自己分析一家公司

## Module 4：行业研究与个股深度

> 核心材料：半导体（圣邦股份、纳芯微、晶方科技、长电科技）、电子（鹏鼎控股、奕瑞科技）、医疗（医美、骨科）、其他（力量钻石、快手、时代电气、锐明科技）

| 行业 | 重点公司 | 学习价值 |
|------|----------|----------|
| 半导体 | 圣邦股份（模拟芯片）、纳芯微（信号链）、晶方科技（封装）、长电科技（封测） | 学会分析芯片全产业链 |
| 电子制造 | 鹏鼎控股（PCB龙头）、奕瑞科技（X射线探测器） | 学会分析制造业壁垒 |
| 医疗 | 医美产业链、骨科材料、塑形市场 | 学会分析高毛利消费医疗 |
| 其他 | 力量钻石（培育钻石）、快手（互联网平台）、时代电气（轨道交通）、锐明技术（商用车） | 跨行业分析能力 |

## 学习路径：5级进阶

| 等级 | 核心内容 | 对应Module | 产出 |
|------|----------|-----------|------|
| L1 认知入门 | 钱是什么、投资vs投机、富人思维 | Module 1 | 读完小狗钱钱+聪明的投资者第一章 |
| L2 方法掌握 | 资产配置、公司分析框架、交易规则 | Module 2+3+5 | 掌握配置方法论和公司分析框架 |
| L3 实践应用 | 自己做配置方案+分析1家公司+建仓 | Module 3+4+5 | 完成自己的配置方案+一份公司分析报告 |
| L4 系统思维 | 打通行业分析、宏观周期、配置策略、交易规则 | Module 2+4+5 | 写出一份完整的投资策略书 |
| L5 融会贯通 | 能教别人、能帮别人做配置、能输出内容 | Module 1-6 | 独立输出内容+帮别人做资产配置 |
