# 一页纸总结：读书会运营与内容输出SOP

## 一句话定义
> 读书会不只分享知识，而是帮一群人建立"从学到做"的闭环——学的会、用得上、赚得到。内容输出是你建立信任、吸引同频人的放大器。

## 核心概念
1. **读书会运营四步法**：选题→共读→讨论→行动——不止读书，更要做
2. **从哪里来**：小红书/公众号/朋友圈引流→知识星球承接→读书会深度转化
3. **怎么留住人**：每次有产出（一句话收获+行动清单）、有互动（讨论+答疑）、有反馈（下次复盘）
4. **内容选题方法论**：
   - 热点话题（当前市场讨论的）
   - 痛点话题（用户最困惑的）
   - 经典话题（经得起时间考验的）
5. **内容形式**：公众号长文→深度；小红书短文→流量；朋友圈→日常触达
6. **投资陪伴的核心**：跌了给信心（不是给结论），涨了给冷静（不是给建议）

## 常见错误
| 错误 | 表现 | 后果 |
|------|------|----------|
| 只读书不做 | 读完就完了，没有行动 | 学员没改变，不会续费 |
| 内容太深 | 一上来讲复杂的投资理论 | 小白听不懂，放弃 |
| 只讲不练 | 分享知识但不带大家实操 | 学了不会用 |
| 没有SOP | 每次临时准备、质量不稳定 | 坚持不下去 |

## 检查清单
- [ ] 我的读书会有明确的目标和流程吗？
- [ ] 我每次活动后都布置了"可执行的行动"吗？
- [ ] 我的内容是针对哪个阶段的用户的？
- [ ] 我有固定的内容输出节奏吗？（每周几篇？发在哪？）
- [ ] 我有没有收集学员的反馈并改进？

## 5个自测题
Q1：读书会运营的四步法是什么？
A：选题→共读→讨论→行动。

Q2：内容选题的三个方向是什么？
A：热点话题、痛点话题、经典话题。

Q3：投资陪伴中"跌了给信心"——信心具体怎么给？
A：不是告诉他"会涨的"（这是预测），而是告诉他"这是正常的"（这是历史规律）+"你的配置扛得住"（这是风控）。

Q4：你目前的内容输出节奏是什么样的？
A：（开放题）找到自己的节奏。

Q5：如果你只能选择一种内容形式（公众号/小红书/朋友圈）持续输出，你选哪个？为什么？
A：（开放题）根据自己的优势选择。
