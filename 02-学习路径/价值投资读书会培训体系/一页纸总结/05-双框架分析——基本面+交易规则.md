# 一页纸总结：双框架分析——基本面+交易规则

## 一句话定义
> 好的投资=好公司（基本面）+好价格（交易规则）——基本面回答"值不值得买"，交易规则回答"什么时候买"。

## 核心概念
1. **框架一：基本面分析**——这家公司优秀吗？
   - 行业好、公司好、管理层好、护城河深、财务健康
   - 结论：它是"可以投资的好公司"还是"不碰"

2. **框架二：交易规则**——现在该买吗？
   - 估值温度低吗？价格有安全边际吗？
   - 建仓节奏怎么安排？分批进还是等？
   - 止损/止盈条件设了吗？

3. **两者关系**：好公司+好价格=买入；好公司+差价格=等待；差公司=不碰
4. **双框架避免的错误**：好公司但买贵了（套牢多年）、坏公司但便宜（价值陷阱）

## 真实案例
- 你的《力量钻石_双框架分析》：基本面（培育钻石龙头、高增长）+交易规则（当时温度多少、建议建仓节奏）
- 你的《奕瑞科技_双框架分析》：基本面（X射线探测器龙头、医疗+工业双驱动）+交易规则（估值区间、建仓策略）

## 常见错误
| 错误 | 表现 | 后果 |
|------|------|----------|
| 只看基本面 | "好公司"不问价格就买 | 买在山顶，套牢 |
| 只看价格 | "便宜"不问基本面 | 价值陷阱 |
| 没有交易计划 | 看好就全仓冲 | 买在不对的位置 |
| 不分批建仓 | All in | 一跌就慌 |

## 检查清单
- [ ] 我做完基本面分析了吗？（好公司？）
- [ ] 我做完交易规则检查了吗？（好价格？）
- [ ] 我的建仓计划写下来了吗？
- [ ] 如果我判断错了，我有止损计划吗？

## 5个自测题
Q1：双框架分析的两个框架分别是什么？
A：基本面分析（值不值得买）+交易规则（什么时候买）。

Q2：好公司+好价格、好公司+差价格、差公司应该分别怎么做？
A：好+好→买入；好+差→等待；差公司→不碰。

Q3：为什么只看基本面"好公司"还不够？
A：再好的公司，买贵了也可能亏很多年（2000年买微软）。

Q4：用双框架分析你最近的一笔投资。
A：（开放题）看你的决策逻辑是否完整。

Q5：双框架分析中你最容易忽略的是哪个部分？
A：（开放题）大多数人忽略交易规则——只关心"买什么", 不关心"什么时候买"。
