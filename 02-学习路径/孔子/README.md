# 孔子学习路径：5级进阶体系

## Level 1：认知入门
核心认知：孔子是中国文化的奠基人，其思想影响了东亚文明两千年。核心是「仁」与「礼」，但更重要的是《论语》中的实践智慧。

## Level 2：方法掌握
- 仁：己所不欲勿施于人
- 学思并重：学而不思则罔
- 三省吾身：每日自我反省
- 中庸之道：过犹不及
- 正名：名不正则言不顺

## Level 3：实践应用
- **学习**：学而时习之、温故知新
- **管理**：其身正不令而行
- **社交**：己所不欲勿施于人、三人行必有我师
- **修身**：吾日三省吾身、欲速则不达

## Level 4：系统思维
- 仁义礼智信的内在体系
- 修身齐家治国平天下的层次逻辑

## Level 5：融会贯通
- 孔子×芒格：学思并重×多元思维、三省吾身×双轨分析
- 孔子×毛主席：实事求是×知之为知之
- 孔子×曾国藩：修身路径对照
