# 一页纸总结：仁与学

## 一句话定义
> 仁是爱人，学是求智。仁+学=完整的人格修养。

## 核心概念
1. **仁的核心**：己所不欲勿施于人（推己及人）
2. **学的方法**：学而不思则罔，思而不学则殆（学思并重）
3. **省的工具**：吾日三省吾身（每日自省）
4. **行的标准**：其身正不令而行（以身作则）

## 真实案例
- 管理者要求团队加班自己先走？其身不正虽令不从
- 跟人争论时想「己所不欲」就能冷静一半
- 学了一堆课生活没变化？因为思而不学

## 常见错误
| 错误 | 表现 |
|------|------|
| 仁而不学 | 人好但认知低 |
| 学而不仁 | 能力强但没有同理心 |
| 只省不改 | 每天反思但不行动 |

## 检查清单
- [ ] 今天的行为有没有违背己所不欲勿施于人？
- [ ] 今天有没有学到新东西并深入思考了？
- [ ] 今天有没有反省自己的过错？
- [ ] 我的言行一致吗？
