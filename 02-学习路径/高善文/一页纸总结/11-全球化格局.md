# 一页纸总结：全球化格局

## 一句话定义
> 全球化的本质是资源在全球的优化配置。退全球化不是在回到过去，是走向一个区域化的新格局。

## 核心概念
1. **全球化的驱动力**：技术进步、贸易自由化、资本流动
2. **退全球化的原因**：地缘政治、贫富差距、产业链安全
3. **区域化的新格局**：从全球供应链到区域供应链
4. **中国的新定位**：从世界工厂到技术升级+内需驱动
5. **对投资的影响**：产业链重构会带来新的投资机会和风险

## 真实案例
- 中美贸易摩擦→产业链从中国向东南亚转移
- 芯片出口管制→中国加速自主可控

## 常见错误
| 错误 | 表现 |
|------|------|
| 认为全球化会完全倒退 | 全球化不会消失，只是形态会变 |
| 忽视中国的调整能力 | 低估了中国产业链的深度和宽度 |
| 只看风险不看机会 | 重构期既有风险也有机会 |
