# 一页纸总结：方法与逻辑

## 一句话定义
> 逻辑推理比数据更重要。数据可以说谎，但逻辑不会。先推逻辑，再找数据验证。

## 核心概念
1. **逻辑优先**：先搞清楚因果关系，再找数据支持
2. **交叉验证**：一个数据可能错，多个数据指向同一方向就不会错
3. **情景分析**：列出不同假设下的各种可能结果
4. **区分存量和流量**：存量是已经积累的，流量是正在发生的
5. **区分短期和长期**：短期看政策，长期看结构
6. **自圆其说**：一个好的分析框架应该能解释过去、现在和未来

## 真实案例
- 高善文用产能周期解释了中国2012-2015年的经济持续下行
- 用预期差框架解释股市领先经济的原因

## 常见错误
| 错误 | 表现 |
|------|------|
| 数据导向 | 先看数据再找解释，容易牵强附会 |
| 忽视逻辑 | 数据支持但逻辑说不通，还是做了判断 |
| 过度拟合 | 用一个框架解释所有现象 |
| 不区分存量和流量 | 把两者混在一起导致错误结论 |

## 检查清单
- [ ] 我的结论有坚实的逻辑基础吗？
- [ ] 多个指标是否指向同一方向？
- [ ] 我考虑了不同的假设情景吗？
- [ ] 我的框架能解释过去也能预测未来吗？
