# 一页纸总结：长期增长动力

## 一句话定义
> 经济增长来自技术进步和制度改革。短期政策可以平滑周期但不能改变长期趋势。

## 核心概念
1. **技术进步的长期性**：技术进步是增长的最终源泉
2. **制度改革红利**：市场化改革释放的巨大增长潜力
3. **人口结构的约束**：劳动力减少、老龄化加速约束潜在增速
4. **资本积累的边际递减**：靠投资拉动的增长模式不可持续
5. **全要素生产率**：衡量技术进步和效率改进的关键指标
6. **追赶式增长的空间**：中国人均GDP仍低于发达国家，还有追赶空间

## 真实案例
- 过去40年高增长：市场化改革+人口红利+全球化+技术追赶
- 当前挑战：人口老龄化+技术追赶空间收窄+全球化退潮

## 常见错误
| 错误 | 表现 |
|------|------|
| 混淆短期和长期 | 用短期政策判断长期趋势 |
| 忽视人口影响 | 低估老龄化对增长的影响 |
| 线性外推 | 过去高增长所以未来也会高增长 |
