# 一页纸总结：长期增长动力

## 一句话定义
> 增长来自技术进步和制度改革，不是短期刺激。

## 核心概念
1. **技术进步的长期性**：技术进步是增长的最终源泉
2. **制度改革的红利**：市场化改革释放的巨大增长潜力
3. **人口结构的约束**：劳动力减少、老龄化加速约束潜在增速
4. **资本积累的边际递减**：靠投资拉动的增长模式不可持续
5. **全要素生产率(TFP)**：衡量技术进步和效率改进的关键指标

## 真实案例
- 中国过去40年的高增长：市场化改革+人口红利+全球化+技术追赶
- 当前挑战：人口老龄化+技术追赶空间收窄+全球化退潮

## 常见错误
| 错误 | 表现 |
|------|------|
| 忽视产能周期 | 不看供给端只盯着需求端 |
| 无视货币时滞 | 货币放松了以为物价马上涨 |
| 把预期当现实 | 市场涨了以为经济马上好 |
| 只看政策不看结构 | 以为政策可以改变长期趋势 |
