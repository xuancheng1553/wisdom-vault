# 一页纸总结：资产价格与预期

## 一句话定义
> 股市是经济预期的晴雨表，不是经济的晴雨表。

## 核心概念
1. **预期差**：市场预期和实际结果之间的差距决定价格波动
2. **领先指标**：股市领先经济6-12个月
3. **预期形成**：预期基于信息和逻辑，不是随机
4. **泡沫的形成**：预期自我强化，脱离基本面
5. **逆向思维**：大多数人乐观时谨慎，大多数人悲观时寻找机会

## 真实案例
- 2022年底市场极度悲观→2023年一季度反弹→经济数据还没改善但股市先涨了
- 2021年初市场极度乐观→核心资产泡沫→后续两年持续调整

## 常见错误
| 错误 | 表现 |
|------|------|
| 忽视产能周期 | 不看供给端只盯着需求端 |
| 无视货币时滞 | 货币放松了以为物价马上涨 |
| 把预期当现实 | 市场涨了以为经济马上好 |
| 只看政策不看结构 | 以为政策可以改变长期趋势 |
