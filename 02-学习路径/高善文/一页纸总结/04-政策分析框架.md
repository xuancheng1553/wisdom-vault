# 一页纸总结：政策分析框架

## 一句话定义
> 理解政策的逻辑比预测具体措施更重要。

## 核心概念
1. **政策目标**：稳增长+防风险+促改革，三者往往冲突
2. **政策工具**：财政、货币、产业、监管
3. **政策传导**：政策→市场预期→经济行为→经济结果
4. **政策边界**：政策不是万能的，长期趋势由结构性因素决定
5. **高善文的方法**：从政策目标倒推政策方向

## 真实案例
- 2021年「双碳」政策→煤炭价格暴涨→政策纠偏→价格回落
- 2022年房地产政策转向→从收紧到支持→市场预期改变→相关股票反弹

## 常见错误
| 错误 | 表现 |
|------|------|
| 忽视产能周期 | 不看供给端只盯着需求端 |
| 无视货币时滞 | 货币放松了以为物价马上涨 |
| 把预期当现实 | 市场涨了以为经济马上好 |
| 只看政策不看结构 | 以为政策可以改变长期趋势 |

## 检查清单
- [ ] 我有从产能、货币、预期、政策、长期五个维度分析吗？

- [ ] 我区分了短期波动和长期趋势吗？
- [ ] 我考虑了预期的反作用吗？
