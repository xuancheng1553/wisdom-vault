# 一页纸总结：通货膨胀与货币

## 一句话定义
> 通胀是货币现象，但货币到物价传导有时滞。

## 核心概念
1. **货币先行**：M2增速变化领先CPI变化6-12个月
2. **通胀结构**：CPI和PPI的差异反映了产业链不同环节的压力
3. **货币传导机制**：货币→信贷→投资→产出→物价
4. **通胀的市场影响**：通胀预期影响资产价格
5. **高善文的方法**：先看货币再看物价，中间看生产端

## 真实案例
- 2020年货币宽松→M2增速上升→2021年PPI大幅上涨→2022年CPI温和上升
- 2021年PPI高企→下游企业成本压力大→利润被挤压→股价下跌

## 常见错误
| 错误 | 表现 |
|------|------|
| 忽视产能周期 | 不看供给端只盯着需求端 |
| 无视货币时滞 | 货币放松了以为物价马上涨 |
| 把预期当现实 | 市场涨了以为经济马上好 |
| 只看政策不看结构 | 以为政策可以改变长期趋势 |
