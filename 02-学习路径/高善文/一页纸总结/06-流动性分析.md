# 一页纸总结：流动性分析

## 一句话定义
> 流动性是资产价格的最直接驱动力。钱多的时候所有资产都会涨，钱少的时候所有资产都会跌。

## 核心概念
1. **流动性的来源**：央行货币政策、银行信贷、外资流入
2. **流动性的度量**：M2、社融、利率、shibor
3. **流动性溢价的扩张与收缩**：流动性充裕时，投资者愿意为任何资产支付更高价格
4. **流动性收缩的传导**：利率上升→资产价格下跌→信用收缩→经济下行
5. **不同资产对流动性的敏感度**：成长股>价值股>债券>商品

## 真实案例
- 2020年全球央行放水→所有资产上涨→2022年美联储加息→所有资产下跌
- 流动性收缩时现金为王，扩张时资产为王

## 常见错误
| 错误 | 表现 |
|------|------|
| 忽视流动性 | 只研究基本面不研究资金面 |
| 以为流动性永远宽松 | 没有意识到流动性会周期性变化 |
| 忽略流动性拐点 | 流动性转向了还没调整策略 |
