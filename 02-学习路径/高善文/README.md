# 高善文学习路径：5级进阶体系

## Level 1：认知入门
了解高善文是谁、他的研究方法论、核心思想地图。
**产出**：能说出高善文与其他经济学家的区别、他的方法框架

## Level 2：方法掌握
- 逻辑先行、数据验证的研究方法
- 交叉验证的指标分析框架
- 政策分析框架：目标→工具→影响
- 区分短期波动与长期趋势

## Level 3：实践应用
- 用产能周期分析经济走势
- 用货币-物价框架预判通胀
- 用预期差分析资产价格
- 用政策框架判断市场方向

## Level 4：系统思维
- 打通产能周期、货币周期、房地产周期、债务周期之间的联系
- 理解短期政策与长期结构性因素的关系
- 建立宏观经济的系统分析框架

## Level 5：融会贯通
- 高善文×芒格：逻辑先行×多元思维模型
- 高善文×毛主席：调查研究×宏观经济分析
- 形成自己的宏观经济分析体系
