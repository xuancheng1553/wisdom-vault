# 芒格 × 毛主席 智慧宝库

> 学习两位伟人的智慧，修身齐家，诚意正心，治国平天下。

## 项目愿景

系统收集、整理、提炼**查理·芒格**与**毛泽东**两位思想家的核心智慧，覆盖人生十个关键维度，打造随时可调用的个人智能助理。

## 知识体系

### 01-知识库

- **[芒格](./01-知识库/芒格/)** — 生平、核心思想（格栅理论/误判心理学/能力圈等）、投资哲学、演讲与著作、推荐书单
- **[毛主席](./01-知识库/毛主席/)** — 生平、核心思想（实践论/矛盾论/实事求是/群众路线等）、军事思想、诗词文学、治国理政
- **[交叉智慧](./01-知识库/交叉智慧/)** — 两位思想家方法论的交叉对照与应用

### 02-维度卡片

按人生十大维度组织，每个维度下汇集两位智者的精要智慧：

1. 修身与学习
2. 创业与事业
3. 投资与理财
4. 婚姻与亲密关系
5. 社交与人脉
6. 身体健康
7. 工作方法
8. 领导力与管理
9. 齐家与教育
10. 理想与境界

### 03-智能助理

移动端友好的 Web 应用，可随时随地查阅、搜索、探索智慧内容。在手机浏览器中打开 `03-智能助理/index.html` 即可使用。
