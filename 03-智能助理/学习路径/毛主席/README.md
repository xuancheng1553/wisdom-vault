# 毛选学习路径：5级进阶体系

## 设计理念
基于「二八法则」聚焦毛选1-4卷中最实用的20%内容，设计从入门到融会贯通的5级进阶体系。

---

## Level 1：认知入门
> 目标：建立毛选的宏观认知，了解为什么要学、学什么、怎么学

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 核心认知 | 毛选不是政治教材，而是**实战方法论** | 能用一句话说清毛选对自己的价值 |
| 历史背景 | 1-4卷的时间跨度与历史分期 | 能画出毛选4卷的时间线地图 |
| 核心思想地图 | 实践论、矛盾论、实事求是、群众路线、独立自主五大核心 | 能说出五个核心思想的名字和一句话定义 |
| 常见错误 | 把毛选当历史书读、只读不想、不联系实际 | 避免三大入门误区 |
| 进阶标准 | 对毛选有了整体认知，能说出自己最想深入学习的2-3个方向 | 读完1-2篇最有感的文章 |

## Level 2：方法掌握
> 目标：掌握毛选的**核心方法论**，能用自己的话阐述

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 实践论 | 感性认识→理性认识→再回到实践的三阶段 | 能用案例解释认识的螺旋上升 |
| 矛盾论 | 主要矛盾与次要矛盾、矛盾的主要方面、矛盾转化 | 能用矛盾分析法分析一个实际问题 |
| 实事求是 | 「实事」就是客观存在、「是」就是规律、「求」就是研究 | 能在决策前问自己「实事求是的做法是什么」 |
| 调查研究 | 没有调查就没有发言权、调查十问 | 掌握基本的调研方法 |
| 常见错误 | 概念背得熟但用不出来、学了一个方法就套用一切场景 | 能识别教条主义和经验主义 |
| 进阶标准 | 能用实践论和矛盾论分析工作和生活中的问题 | 写出3篇运用矛盾分析法的笔记 |

## Level 3：实践应用
> 目标：把方法应用到**创业、投资、管理、工作**等具体场景

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 创业应用 | 调研市场（没有调查就没有发言权）、聚焦突破（集中优势兵力）、快速迭代（实践-认识-再实践） | 用毛选方法分析自己的创业/项目 |
| 投资应用 | 辨别主要矛盾（矛盾论）、战略藐视战术重视、不打无准备之仗 | 用矛盾分析法分析一只股票/行业 |
| 管理应用 | 群众路线（从群众中来、到群众中去）、民主集中、解剖麻雀 | 改善自己的管理方式 |
| 工作方法 | 以点带面、典型引路、抓主要矛盾 | 优化每周工作计划 |
| 常见错误 | 照搬战争年代的策略不转化、忽略具体场景的差异 | 能针对不同场景调整方法 |
| 进阶标准 | 在2-3个实际场景中长期运用毛选方法，并有案例复盘 | 有3个以上自己的应用案例 |

## Level 4：系统思维
> 目标：打通毛选各篇章之间的联系，建立系统思维框架

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 跨篇联系 | 实践论↔矛盾论↔实事求是的内在逻辑链 | 能画出毛选方法论之间的关联图 |
| 辩证思维 | 矛盾转化、对立统一、量变到质变 | 能用辩证思维分析复杂问题 |
| 战略战术 | 全局与局部、长期与短期、主要与次要 | 能制定自己的「论持久战」 |
| 体系构建 | 认识论+方法论+实践论的三位一体 | 建立自己的毛选方法体系 |
| 常见错误 | 碎片化学习、只学工具不学思维、理论与实践脱节 | 开始系统性应用 |
| 进阶标准 | 建立了自己的毛选思维体系，能在复杂决策中综合运用 | 完成一篇跨篇章的深度分析文章 |

## Level 5：融会贯通
> 目标：将毛选智慧与其他思想体系交叉印证，形成自己的智慧体系

| 维度 | 内容 | 学习产出 |
|------|------|----------|
| 毛选×芒格 | 实践论↔多元思维模型、矛盾论↔误判心理学、实事求是↔能力圈 | 写一篇芒格与毛主席的对比分析 |
| 毛选×孔子 | 实事求是↔知行合一、调查研究↔学思并重 | 找到儒毛共通的方法论 |
| 毛选×老子 | 矛盾转化↔阴阳辩证、战略藐视↔以柔克刚 | 理解东西方辩证法的异同 |
| 毛选×王阳明 | 实践论↔知行合一、改造世界↔致良知 | 梳理知行关系的演变 |
| 常见错误 | 牵强附会、概念混淆、忽视时代差异 | 能区分不同思想体系的适用范围 |
| 进阶标准 | 形成了自己的智慧体系，能自如调用多个思想资源 | 建立了个人知识库和思维框架 |

---

## 学习建议

1. **不要贪多**：每个Level至少停留1-2周
2. **输出倒逼输入**：每学完一篇，写300字以上的读后感
3. **联系实际**：每个方法都要问自己「这个能用在哪里」
4. **交叉验证**：用芒格的方法检验毛选的结论，反之亦然
