# 交叉智慧：芒格×米勒——如何寻找人生伴侣

## 为什么这两个人放在一起？
芒格提供了**哲学和原则**——你应该怎么想、怎么选。
米勒提供了**科学和方法**——研究者们通过数据发现了什么规律。
两个结合起来，就是你找伴侣的完整操作系统。

## 对照表

| 问题 | 芒格会怎么说 | 米勒会怎么说 |
|------|------------|------------|
| 找伴侣最重要的是什么？ | 选对人比培养人重要 | 选择安全型依恋，持有成长信念 |
| 怎么判断一个人？ | 看他做什么，不是听他说什么 | 观察归因方式：把好的归因于人品还是运气 |
| 什么时候推进关系？ | 第三次约会左右 | 自我表露的互惠性——你分享一点，对方分享一点 |
| 长期关系靠什么？ | 和比你强的人结婚，一起变好 | 爱情三角：亲密+激情+承诺；5:1积极比率 |
| 怎么避免选错？ | 反过来想：什么会毁掉关系，然后避开 | 识别四大骑士和危险信号 |
| 自己的准备？ | 先成为值得被选的人 | 成为安全型，持有成长信念 |

## 共同警告
两位智者有惊人的共识：
1. **感觉不可靠**——芒格说头脑发热危险，米勒说激情会消退
2. **行为比语言真实**——芒格说看做什么，米勒有数据支持
3. **选错比等待代价大**——芒格说宁缺毋滥，米勒说不良关系损害健康
4. **关系需要经营**——芒格说理性是道德责任，米勒说五种维持策略
