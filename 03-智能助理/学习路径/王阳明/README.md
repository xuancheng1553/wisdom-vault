# 王阳明学习路径：5级进阶体系

## Level 1：认知入门
核心认知：王阳明是历史上极少数能「立功立德立言」三不朽的人物——既是军事家又是哲学家。心学的核心是三个词：心即理、知行合一、致良知。

## Level 2：方法掌握
- 心即理：求理不必外求，内观本心
- 知行合一：知和行是一体的，知道不做等于不知道
- 致良知：每个人内心都有良知，听从它
- 事上练：在事情上磨练心性

## Level 3：实践应用
- **修身**：致良知、事上练、静坐
- **管理**：破心中贼难——管理自己的念头
- **学习**：知行合一——学以致用
- **领导力**：心学领导力——以心服人

## Level 4：系统思维
- 心学与程朱理学的对比
- 知行关系的思想史演变

## Level 5：融会贯通
- 王阳明×毛主席：知行合一×实践论
- 王阳明×芒格：致良知×理性道德
- 王阳明×曾国藩：修身路径对照
