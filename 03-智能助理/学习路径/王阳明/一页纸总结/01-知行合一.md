# 一页纸总结：知行合一

## 一句话定义
> 知道而不做等于不知道。知和行是一体的，不可分割。

## 核心概念
1. **知是行之始**：认知是行动的开始
2. **行是知之成**：行动是认知的完成
3. **知行本一体**：被私欲隔断了才分成两件事
4. **事上练**：在具体事情上磨练心性

## 真实案例
- 知道健康重要但不运动 → 不是真知道
- 知道投资需要耐心但不持有 → 不是真知道
- 知道对孩子要有耐心但发火 → 不是真知道
- 毛主席说：读书是学习，使用也是学习

## 常见错误
| 错误 | 表现 |
|------|------|
| 知而不行 | 知道但不做 |
| 行而不知 | 做了不知道为什么成功/失败 |
| 知行为二 | 把学习和实践分开 |

## 检查清单
- [ ] 我「知道」的事情有没有去做？

- [ ] 我做过的事情有没有复盘总结？
- [ ] 最近学到了什么新知识并马上实践了？
