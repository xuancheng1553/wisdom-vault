# 一页纸总结：爱情三角理论

## 一句话定义
> 完美的爱情由亲密（情感连接）、激情（身体吸引）、承诺（决定维持）三个成分构成，三者缺一不可。

## 核心概念
1. **亲密**：情感上的亲近、理解、分享、支持
2. **激情**：身体上的吸引、浪漫、性欲望
3. **承诺**：理性上的决定、对未来的规划、坚守
4. **七种爱情**：无爱（三者皆无）、喜欢（仅亲密）、迷恋（仅激情）、空爱（仅承诺）、浪漫之爱（亲密+激情）、相伴之爱（亲密+承诺）、愚昧之爱（激情+承诺）、圆满之爱（三者兼备）

## 真实案例
- 恋爱初期：激情主导，亲密在建立中，承诺尚未形成
- 结婚多年：激情下降，亲密深厚，承诺稳固
- 危机时刻：亲密受损，激情消退，承诺面临考验

## 常见错误
| 错误 | 表现 |
|------|------|
| 只追激情 | 追求心跳加速，忽略了亲密和承诺 |
| 只有承诺 | 维持一段无爱无趣的关系 |
| 以为爱情不变 | 爱情的形态会随着时间变化 |
