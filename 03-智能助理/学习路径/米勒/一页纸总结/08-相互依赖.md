# 一页纸总结：相互依赖

## 一句话定义
> 亲密关系的满意度取决于奖赏减去成本，但真正维系长期关系的是承诺。

## 核心概念
1. **奖赏**：关系带来的愉悦、支持、陪伴、资源
2. **成本**：关系中付出的时间、精力、妥协、冲突
3. **结果 = 奖赏 - 成本**：差值越大越满意
4. **比较水平**：你觉得自己应得的水平——来自过去经验和社会比较
5. **替代选择**：有没有其他可选的关系——替代选择越少，依赖性越强
6. **承诺**：希望关系继续的主观意愿——不仅是满意，更是承诺在维系长期关系

## 真实案例
- 满意但分手：奖赏够多但替代选择更好
- 不满意但不分手：替代选择太少（经济依赖/孩子/社会压力）
- 最稳固的关系：满意 + 高承诺 + 低替代选择

## 常见错误
| 错误 | 表现 |
|------|------|
| 只看奖赏不看成本 | 觉得爱情不需要计算 |
| 比较水平过高 | 拿偶像剧标准要求现实关系 |
| 低估承诺的作用 | 以为满意就够，遇到困难就放弃 |

## 检查清单
- [ ] 这段关系给我的奖赏是什么？
- [ ] 我付出的成本是什么？值得吗？
- [ ] 我的比较水平合理吗？
- [ ] 除TA之外我还有其他选择吗？
- [ ] 我们之间的承诺有多强？
