# 一页纸总结：沟通

## 一句话定义
> 高质量沟通的核心是精确表述、积极倾听、保持礼貌和确认对方。沟通不良是关系破裂的首要原因。

## 核心概念
1. **精确表述**：用第一人称表达感受（我感到…）而非指责（你总是…）
2. **积极倾听**：专注听对方说，不打断、不预判、不急着反驳
3. **礼貌和确认**：尊重对方的表达，确认自己理解正确
4. **要求/退避模式**：一方追一方逃，最伤害关系的沟通模式
5. **修复尝试**：冲突后主动示好、道歉或幽默化解

## 真实案例
- ❌：你从来不洗碗！（批评人格）→ 我没不洗！（辩护）→ 算了不说了（冷战）
- ✅：我最近有点累，今晚你可以洗碗吗？（温和表达）→ 好的，我来洗（积极回应）

## 常见错误
| 错误 | 表现 |
|------|------|
| 批评人格 | 你从来/你总是（攻击人而非行为） |
| 被动沟通 | 不说出来，指望对方猜 |
| 消极沉默 | 不说话等于解决问题？不，等于积累问题 |
| 读心术 | 以为对方应该知道你在想什么 |

## 检查清单
- [ ] 我用的是一人称还是二人称？
- [ ] 我是在表达感受还是在指责？
- [ ] 对方说话时我在认真听还是想着怎么反驳？
- [ ] 吵架后我有尝试修复吗？

Q3：修复尝试有哪些形式？A：道歉、幽默、表达理解、示好。
