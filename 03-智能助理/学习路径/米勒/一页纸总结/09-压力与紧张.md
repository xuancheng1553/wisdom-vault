# 一页纸总结：压力与紧张

## 一句话定义
> 嫉妒、欺骗、背叛是亲密关系中最常见的压力来源，处理不当会严重损害信任。

## 核心概念
1. **嫉妒类型**：反应性嫉妒（真的发生了）vs 怀疑性嫉妒（只是怀疑）
2. **嫉妒的成因**：依赖关系 + 替代选择威胁 + 不安全感
3. **欺骗**：善意的谎言（不想伤害对方）vs 利己的欺骗（为自己获益）
4. **背叛**：违反关系规范的行为——不一定出轨才是背叛
5. **信任修复**：承认→道歉→解释→补偿→改变（需要时间和一致性）

## 真实案例
- 怀疑性嫉妒：TA不接电话 → 肯定在和别人在一起 → 查手机 → 引发争吵 → 关系恶化
- 善意的谎言：我觉得你穿这个很好看（其实觉得一般）
- 利己的欺骗：我加班呢（其实在和朋友玩）

## 常见错误
| 错误 | 表现 |
|------|------|
| 过度嫉妒 | 变成控制行为，把对方推得更远 |
| 欺骗的滑坡 | 从小谎到大谎 |
| 信任破坏了不修复 | 以为时间会冲淡一切 |
| 把善意的谎言当成没问题 | 长期累积也会损害关系 |

## 检查清单
- [ ] 我的嫉妒是合理的还是过度猜测？
- [ ] 我最近有没有对伴侣说谎？动机是什么？
- [ ] 信任被破坏后，我有没有主动修复？
- [ ] 我给了伴侣足够的安全感吗？
