# 黄帝内经学习路径：5级进阶体系

## Level 1：认知入门
核心认知：黄帝内经不是一本医书，而是一套关于「人如何与自然和谐相处」的生命哲学体系。

## Level 2：方法掌握
- 阴阳：对立统一的基本规律
- 五行：木火土金水的相生相克
- 四气调神：春生夏长秋收冬藏
- 正气存内邪不可干：免疫力
- 上工治未病：预防大于治疗

## Level 3：实践应用
- **饮食**：食饮有节、七分饱、顺应四时
- **作息**：跟着太阳走、子午觉
- **运动**：常欲小劳莫大疲
- **情志**：怒伤肝喜伤心——情绪管理
- **四季**：春养肝、夏养心、秋养肺、冬养肾

## Level 4：系统思维
- 藏象学说（五脏六腑的功能体系）
- 经络学说（人体的能量网络）
- 辨证论治（因人因时因地制宜）

## Level 5：融会贯通
- 黄帝内经×芒格：系统思维×多元模型
- 黄帝内经×老子：道法自然×顺应四时
- 黄帝内经×现代医学：中医的当代价值
