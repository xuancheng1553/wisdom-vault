# 本杰明·富兰克林学习路径：5级进阶体系

## Level 1：认知入门
核心认知：富兰克林是美国版的「曾国藩」——一个通过系统化自我修炼成就非凡的人。他的十三条美德体系是历史上最早的自我管理方法之一。

## Level 2：方法掌握
- 十三条美德的系统化修炼方法
- 每周聚焦一项、十三周循环的节奏
- 时间管理方法
- 节俭和积累的财富观

## Level 3：实践应用
- **修身**：用美德表格每日自省
- **工作**：时间块管理法
- **理财**：节俭积累 + 投资
- **社交**：诚实和信誉是最重要的社交货币

## Level 4：系统思维
- 美德体系的内在逻辑（从节制到谦逊的递进）
- 修身与成事的关系

## Level 5：融会贯通
- 富兰克林×芒格：十三条美德×误判心理学
- 富兰克林×曾国藩：中西修身对照
