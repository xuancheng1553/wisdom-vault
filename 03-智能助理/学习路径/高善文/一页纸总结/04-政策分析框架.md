# 一页纸总结：政策分析框架

## 一句话定义
> 理解政策的逻辑比预测政策的具体措施更重要。从政策目标倒推政策工具。

## 核心概念
1. **政策目标**：稳增长+防风险+促改革，三者往往冲突
2. **政策排序**：不同时期政策目标的优先级不同，决定政策走向
3. **政策工具**：财政、货币、产业、监管，各有各的传导路径
4. **从目标倒推**：先想政府想做什么，再想它会用什么工具
5. **政策的边界**：政策不是万能的，不能改变长期趋势

## 真实案例
- 2021年双碳政策→煤炭暴涨→政策纠偏→价格回落
- 2022年房地产政策转向→从收紧到支持→市场预期改变→股票反弹

## 常见错误
| 错误 | 表现 |
|------|------|
| 只看工具不看目标 | 分析具体政策但不理解背后的意图 |
| 高估政策效果 | 以为政策可以改变一切 |
| 忽视政策时滞 | 政策出了以为效果马上来 |

## 检查清单
- [ ] 当前政策的首要目标是什么？（稳增长？防风险？）
- [ ] 我理解政策背后的逻辑了吗？
- [ ] 政策落地需要多久才能看到效果？
- [ ] 政策的边界在哪里？

Q1：怎么从政策目标倒推政策工具？
Q2：当前的政策是宽松还是收紧？怎么看出来的？
Q3：政策对市场有什么影响？
Q4：什么时候说政策已经见顶或见底？
Q5：用政策分析框架分析当前的一个政策变化。
