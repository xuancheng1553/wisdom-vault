# 一页纸总结：产能周期

## 一句话定义
> 中国经济本质上是产能周期驱动。产能过剩时下行，出清后复苏。

## 核心概念
1. **产能过剩**：投资过多导致供给超过需求，价格下跌、利润压缩
2. **产能出清**：亏损企业退出，供给减少，价格回升
3. **产能周期**：从过剩到出清再到扩张的完整循环
4. **中国特色的产能波动**：政府的产业政策会放大周期波动
5. **产能周期与投资时机**：在出清末期布局，在过剩高峰期回避

## 真实案例
- 2009-2011年的大规模刺激→产能过剩→2012-2015年的经济下行→2016-2017年的去产能→经济企稳
- 你的投资：在产能出清末期布局相关行业，在产能过剩高峰期回避

## 常见错误
| 错误 | 表现 |
|------|------|
| 忽视产能周期 | 不看供给端只盯着需求端 |
| 无视货币时滞 | 货币放松了以为物价马上涨 |
| 把预期当现实 | 市场涨了以为经济马上好 |
| 只看政策不看结构 | 以为政策可以改变长期趋势 |
