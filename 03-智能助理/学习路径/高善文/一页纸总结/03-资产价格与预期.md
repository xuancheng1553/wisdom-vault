# 一页纸总结：资产价格与预期

## 一句话定义
> 股市不是经济的晴雨表，是经济预期的晴雨表。预期变了，价格先变，经济后变。

## 核心概念
1. **预期差**：市场预期与实际结果之间的差距决定价格波动
2. **领先关系**：股市领先经济6-12个月
3. **预期的形成**：基于信息和逻辑，不是随机的
4. **自我实现**：预期会反过来影响行为，行为影响结果
5. **反向思维**：大多数人乐观时谨慎，大多数人悲观时寻找机会
6. **流动性驱动短期**：短期看流动性，长期看基本面

## 真实案例
- 2022年底市场极度悲观→2023年一季度反弹→经济数据还没改善但股市涨了
- 2021年初极度乐观→核心资产泡沫→后续两年持续调整

## 常见错误
| 错误 | 表现 |
|------|------|
| 把预期当现实 | 市场涨了以为经济会马上好 |
| 线性外推 | 过去涨所以未来还会涨 |
| 忽视预期差 | 只看数据和事实，不看市场怎么想 |
