# 一页纸总结：80-20法则（帕累托法则）

## 一句话定义
> 20%的投入产生80%的结果。要聚焦在那20%的关键因素上，其他的可以少花精力。

## 核心概念
1. **二八分布**：大多数事物不是平均分布的，少数关键因素决定了大部分结果
2. **投入产出不平衡**：80%的产出来自20%的投入，80%的问题来自20%的原因
3. **聚焦法则**：找到那20%的关键因素，把80%的精力放在上面
4. **不要追求完美**：剩下的80%投入只贡献20%的结果，不值得花太多时间
5. **动态变化**：那20%的关键因素不是固定的，需要定期重新评估

## 真实案例
- 80%的收入来自20%的客户 → 要服务好核心客户
- 80%的投资收益来自20%的决策 → 芒格说一生只需做少数几个重大决策
- 你的3600元群的收入来自20%的内容 → 把那20%的内容做得更好
- 80%的学习效果来自20%的书 → 把那几本最好的书读100遍

## 常见错误
| 错误 | 表现 |
|------|------|
| 平均用力 | 所有事情花同样的时间和精力 |
| 找到20%但不行动 | 知道应该聚焦，但舍不得放弃其他80% |
| 忽视那20%的变化 | 那20%的关键因素不是一成不变的 |
| 追求完美主义 | 在80%不重要的事上追求完美 |

## 检查清单
- [ ] 我当前最重要的事是哪一件？（那20%）
- [ ] 我有没有在不重要的事上花太多时间？
- [ ] 我的精力分配是二八分布还是平均分布？
- [ ] 那20%的关键因素最近变了没有？
