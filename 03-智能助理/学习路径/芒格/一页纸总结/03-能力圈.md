# 一页纸总结：能力圈

## 一句话定义
> 清楚地知道自己的知识边界，在圈内行动，在圈外学习。

## 核心概念
1. **圈内 vs 圈外**：你真正懂的领域 vs 你只是听说过的领域
2. **边界意识**：知道自己知道什么，更要知道自己不知道什么
3. **拓展原则**：能力圈可以扩大，但只能通过系统学习逐步扩大
4. **红队演练**：让一个懂行的人质疑你的判断

## 真实案例
- ❌ 看到生物科技涨得好就买入 → 你不懂这个行业，退出能力圈
- ✅ 只投自己熟悉行业的公司 → 知道自己懂什么
- ❌ 别人推荐一只股票，你因为信任TA就买入 → 你在用别人的能力圈代替自己的

## 常见错误
| 错误 | 表现 |
|------|------|
| 能力圈错觉 | 读过几篇文章就觉得懂了 |
| 他人代圈 | 跟着所谓的大V投资 |
| 边界模糊 | 知道个大概就当懂了 |
| 不敢拓展 | 永远只在自己的小圈子里 |

## 检查清单
- [ ] 这个决策在我的能力圈内吗？
- [ ] 有人能挑战我的判断而我能回应吗？
- [ ] 我能不能用10分钟给一个外行讲清这件事？
- [ ] 我是真的懂，还是只是听说过？
