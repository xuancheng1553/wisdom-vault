# 一页纸总结：从错误中学习

## 一句话定义
> 聪明人从自己的错误中学习，智慧的人从别人的错误中学习。研究失败比研究成功更有价值。

## 核心概念
1. **失败清单**：芒格一生都在收集各种失败案例
2. **研究死亡比研究活法更重要**：知道怎么死，避开死路，自然就活着
3. **反面证据优先**：做决定之前先找反面证据
4. **错误是学习机会**：不是需要掩盖的耻辱
5. **别人的错误是免费教材**：不需要自己亲身撞南墙

## 真实案例
- 芒格研究的经典失败案例：科维特（高速扩张导致崩盘）、所罗门兄弟（内控失败）、长期资本管理公司（高杠杆崩盘）
- 把失败案例整理成清单，每次做决策前对照检查
- 你的创业：从99元群到3600元群的迭代本身就是在从错误中学习

## 常见错误
| 错误 | 表现 |
|------|------|
| 掩盖错误 | 犯错后假装没发生 |
| 重复犯错 | 没有复盘，同样的错误反复犯 |
| 只从自己错中学 | 明明可以看别人的案例避免 |
| 过度反思 | 为每个小错过度自责 |

## 检查清单
- [ ] 我最近犯的一个错误是什么？我从中学到了什么？
- [ ] 我有没有记录错误的习惯？
- [ ] 我有没有研究过别人在这个领域的失败案例？
- [ ] 我现在的决定中有没有别人已经犯过的错误？
