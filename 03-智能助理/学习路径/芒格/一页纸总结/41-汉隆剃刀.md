# 一页纸总结：汉隆剃刀

## 一句话定义
> 能用愚蠢解释的，就不要归因于恶意。别人做了让你不舒服的事，大概率不是针对你。

## 核心概念
1. **归因的选择**：同一件事可以归因为恶意，也可以归因为愚蠢、疏忽、无知
2. **节省情绪成本**：归因于恶意会让你愤怒，归因于愚蠢你只会摇摇头
3. **自我保护**：大多数人的大多数行为是以自己为中心的，不是在针对你
4. **投资中的应用**：股价下跌不是因为市场针对你，是市场在理性调整
5. **不是什么都归因于愚蠢**：重复的恶意行为就不能再用汉隆剃刀

## 真实案例
- 对方没回信息：不是因为讨厌你，可能只是在忙
- 股市跌了：不是市场在针对你，是各种因素综合结果
- 别人忘了你的事：不是不重视你，是太忙了

## 常见错误
| 错误 | 表现 |
|------|------|
| 过度归因于恶意 | 把一切不舒服都当成别人的针对 |
| 汉隆剃刀当做绊脚石 | 该严肃对待的问题也归因于愚蠢 |
| 滥用剃刀 | 重复同样的伤害还找借口 |

## 检查清单
- [ ] 我是不是在把别人的疏忽归因于恶意？
- [ ] 对方有没有可能是无意的？
- [ ] 这个模式是第一次发生还是反复发生？
