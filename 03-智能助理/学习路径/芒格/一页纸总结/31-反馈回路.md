# 一页纸总结：反馈回路

## 一句话定义
> 行为的结果会影响未来的行为。正反馈让趋势加速，负反馈让系统稳定。

## 核心概念
1. **正反馈回路**（增强回路）：A导致B，B反过来增强A。如：赚更多钱→投资更多→赚更多钱
2. **负反馈回路**（平衡回路）：偏差越大，修正力越强。如：房间太热→开空调→房间变冷
3. **反馈时滞**：反馈往往有延迟，不注意时滞会导致过度修正
4. **识别回路类型**：判断你面对的是正反馈还是负反馈，决定你怎么应对
5. **临界点**：正反馈回路会让系统加速到临界点后失控

## 真实案例
- 你的社群的正反馈：内容越好→口碑越好→用户越多→内容越好
- 股市的正反馈：涨→更多人买→更涨→更多人买（直到泡沫破裂）
- 个人成长的负反馈：过度工作→效率下降→工作更慢→更努力（恶性循环）
- 芒格的阅读习惯：读得越多→知道得越多→决策越好→赚得越多→有更多时间读

## 常见错误
| 错误 | 表现 |
|------|------|
| 忽略正反馈陷阱 | 以为一直涨是正常的，忽视了泡沫风险 |
| 忽视时滞 | 做了调整没看到效果就继续调整 |
| 混淆正负反馈 | 该加强的时候削弱，该削弱的时候加强 |
| 没有及时介入 | 正反馈回路已经往坏的方向走了也不干预 |
