# 一页纸总结：如何寻找伴侣——芒格×米勒综合路径

## 一句话定义
> 先成为值得被选的人，用依恋类型和价值观筛选，观察行为而非语言，在第三次约会前推进关系，用爱情三角评估长期潜力。

---

## 五步路径（芒格 × 米勒综合）

### 第一步：准备自己——成为值得被选的人

**芒格说：** 「找到一位好配偶的最佳办法是你先成为一位好配偶。」
**米勒说：** 吸引力遵循匹配现象——人们倾向于选择和自己整体吸引力相当的人。

| 你的准备清单 | 说明 |
|------------|------|
| 建立你的能力圈和事业 | 芒格：你的认知高度决定你吸引的人的质量 |
| 了解自己的依恋类型 | 米勒：安全型最能建立健康关系。如果你是恐惧型/焦虑型/回避型，先修复自己 |
| 从宿命信念转向成长信念 | 米勒：持有成长信念的人更愿意为关系投入，遇到问题积极解决 |
| 建立你的声誉 | 芒格：信任是最重要的社交货币 |
| 成为一个安全型的人 | 能亲密也能独立，能信任也能被信任 |

---

### 第二步：筛选——用标准而不是感觉

**不要只看感觉。芒格和米勒都认为：感觉是最不可靠的决策依据。**

**硬标准清单（两个理论都验证过）：**

| 标准 | 来自 | 验证方法 |
|------|------|---------|
| 安全型依恋 | 米勒 | 观察TA能否亲密也能独立，情绪是否稳定 |
| 成长信念 | 米勒 | TA认为关系可以改善还是「合不来就算了」 |
| 价值观一致 | 芒格+米勒 | 相似性中价值观相似比兴趣相似更重要 |
| 行为可靠 | 芒格 | 说到做到，压力下的行为 |
| 处理冲突的方式 | 米勒 | 观察分歧时TA是温和开始还是攻击 |
| 比你强让你变得更好 | 芒格 | 芒格说这是最重要的择偶标准 |

**危险信号（发现任一就应谨慎）：**

| 信号 | 来自 | 为什么危险 |
|------|------|-----------|
| 批评/鄙视/辩护/冷战 | 戈特曼/米勒 | 末日骑士出现，长期必毁关系 |
| 要求/退避模式 | 米勒 | 一方追一方逃，恶性循环 |
| 焦虑或回避型 | 米勒 | 两种不安全的依恋模式都容易导致关系问题 |
| 言行不一 | 芒格 | 行为比语言诚实，不一致说明有问题 |
| 嫉妒或控制 | 米勒+芒格 | 过度的嫉妒是控制的前兆 |

---

### 第三步：推进——第三次约会法则

**芒格的节点：** 第三次约会左右是关系推进的关键窗口。
**米勒的理论：** 自我表露的互惠性——你分享一点，对方也分享一点，亲密感逐步加深。

**第三次约会要做的事：**
1. 主动分享一次真实的内心（不是你的经历，是你对某事的感受）
2. 问一个价值观问题（「你理想的生活是什么样的？」）
3. 观察TA的反应——是接住还是回避

**不要做的事：**
- ❌ 暧昧太久不推进（米勒：关系会自然消散）
- ❌ 靠活动填充关系（一起玩不等于关系深，没有自我表露就没有亲密）
- ❌ 假装不是自己（芒格：声誉是最重要的资产，诚实就是最好的策略）

---

### 第四步：评估——用爱情三角打分

**用米勒的爱情三角理论，给关系打三个分：**

| 成分 | 1-10分 | 理想长期关系 |
|------|--------|-------------|
| 亲密（情感连接/分享内心） | ≥7分 | 你能和TA说真话，TA能接住 |
| 激情（身体吸引/浪漫） | ≥6分 | 长期关系激情会下降但不应归零 |
| 承诺（决定维持/未来规划） | ≥7分 | TA愿意为关系投入和规划 |

**只有当亲密和承诺都在7分以上、激情在6分以上，才值得进入长期关系。**

---

### 第五步：长期——五种关系维持策略

**米勒的五种维持策略，配合芒格的检查清单：**
1. **积极性**：保持乐观愉快，日常积极互动
2. **开放性**：分享内心世界，定期深度交流
3. **保证**：表达承诺和对关系的重视
4. **社交网络**：建立共同的朋友圈
5. **共享任务**：公平分担生活责任

**芒格的检查清单：**
- [ ] 我还在成长为更好的自己吗？
- [ ] 我们有在共同成长吗？
- [ ] 我们的沟通方式是建设性的还是破坏性的？
- [ ] 我们还在互相选择还是惯性维持？

---

## 筛选流程图

```
认识新的人
  ↓
先看：TA的行为可靠吗？（芒格）
   ↓ 否→ 换下一个
   ↓ 是
再看：TA是安全型依恋吗？（米勒）
   ↓ 否→ 谨慎考虑，不建议长期
   ↓ 是
再看：TA的价值观和我一致吗？（芒格+米勒）
   ↓ 否→ 短期可以，长期大概率出问题
   ↓ 是
再看：TA有危险信号吗？（戈特曼+芒格）
   ↓ 有→ 撤退
   ↓ 无
推进：第三次约会推进关系（自我表露）
  ↓
评估：爱情三角打分
  ↓
决定：进入长期关系 + 五种维持策略
```

## 常见错误

| 错误 | 来自 | 为什么致命 |
|------|------|-----------|
| 头脑发热 | 芒格 | 激情期做的决定，激情退了就后悔 |
| 忽视危险信号 | 米勒+戈特曼 | 早期发现的信号不会自动消失，只会恶化 |
| 急于推进或拖太久 | 芒格+米勒 | 太快是头脑发热，太慢是暧昧消耗 |
| 试图改变对方 | 芒格 | 选对人比培养人重要，这个在关系中也成立 |
| 只用爱情三角的一个成分 | 米勒 | 只有激情的关系最脆弱 |
