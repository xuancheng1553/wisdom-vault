# 一页纸总结：阅读与终身学习

## 一句话定义
> 我见过的所有聪明人没有一个不大量阅读的——没有，一个都没有。

## 核心概念
1. **深度阅读**：几本好书读100遍胜过读100本书
2. **终身不分年龄**：芒格99岁还在每天阅读
3. **跨学科阅读**：不只读自己专业的书
4. **阅读即思考**：阅读不是任务而是生活方式

## 真实案例
- 芒格每天花4-6小时阅读，被称为长着两条腿的书架
- 他说：我这辈子遇到的所有明智之人，没有一个不大量阅读的

## 常见错误
| 错误 | 表现 |
|------|------|
| 只读不思考 | 读了很多但不消化 |
| 浅尝辄止 | 什么书都读但都不深 |
| 毕业就停止 | 把学习当任务不是生活方式

## 检查清单
- [ ] 我今天阅读了吗？
- [ ] 我是深读还是泛读？
- [ ] 我读完有思考和应用吗？
