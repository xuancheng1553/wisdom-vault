# 一页纸总结：冗余与安全边际

## 一句话定义
> 关键系统要留有余地。不要满负荷运转，留出安全空间应对意外。价格是你付出的，价值是你得到的，中间的差距就是安全边际。

## 核心概念
1. **安全边际的内涵**：以低于内在价值的价格买入，留足容错空间
2. **冗余不是浪费**：备用系统看起来是多余的，但关键时候救你一命
3. **对抗不确定性**：未来是不可预测的，安全边际是你唯一的保护
4. **不同场景的应用**：投资需要安全边际，工程设计需要冗余，生活需要备用计划
5. **过度冗余也是问题**：太多安全边际会降低效率

## 真实案例
- 投资安全边际：你评估一家公司值100元，80元买入，即使你错了它值90元，你还有10元的缓冲
- 你的资产配置：每种资产配置一定比例，不把所有钱投到一个地方
- 飞机的冗余设计：多个引擎、多个系统，坏了一个还有备份
- 生活中的备用计划：现金储备、备用方案、职业备份

## 常见错误
| 错误 | 表现 |
|------|------|
| 没有安全边际 | 满仓买入，没有容错空间 |
| 把冗余当浪费 | 觉得备用金用不上就是亏了 |
| 过度冗余 | 留了太多现金，失去了复利机会 |
| 错估安全边际需求 | 表面上有安全边际，实际上没有 |
