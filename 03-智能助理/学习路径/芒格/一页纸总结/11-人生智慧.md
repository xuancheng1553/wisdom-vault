# 一页纸总结：芒格的人生智慧

## 一句话定义
> 不嫉妒、不怨恨、不过度消费、保持乐观、和快乐的人在一起。

## 核心概念
1. **不嫉妒**：嫉妒是最愚蠢的罪，因为它不会给你带来任何快乐
2. **不怨恨**：怨恨是拿别人的错误惩罚自己
3. **不过度消费**：节俭不是吝啬，而是对资源的尊重
4. **保持乐观**：理性乐观主义——看得到问题，但相信能解决
5. **和快乐的人在一起**：情绪是会传染的
6. **做自己感兴趣的事**：热爱是最好的驱动力
7. **终身阅读**：每天变聪明一点点

## 真实案例
- 芒格99岁还在每天阅读和思考，他的长寿哲学极其简单
- 早年经历离婚、丧子、失明等打击，但他从未被击垮
- 他一生节俭，虽然有巨额财富但生活朴素

## 常见错误
| 错误 | 表现 |
|------|------|
| 攀比 | 总在和别人比，永远不满足 |
| 怨恨 | 把精力浪费在过去的伤害上 |
| 过度消费 | 靠花钱获得短暂快乐 |
| 抱怨 | 把时间和精力花在不能改变的事情上 |

## 检查清单
- [ ] 我今天有没有嫉妒别人？
- [ ] 我有没有在怨恨一些无法改变的事？
- [ ] 我今天的消费是必要的还是冲动的？
- [ ] 我有没有花时间做自己真正感兴趣的事？
- [ ] 我有没有阅读和学习？
- [ ] 我和什么人在一起？他们的情绪在影响我吗？
