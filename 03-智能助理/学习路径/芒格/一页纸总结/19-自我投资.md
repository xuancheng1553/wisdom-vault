# 一页纸总结：自我投资

## 一句话定义
> 20岁最应该投资的是你自己的大脑。认知提升了，赚钱是自然的结果。

## 核心概念
1. **投资大脑**：20岁最重要的投资不是股票是自己
2. **技能>金钱**：能力带来收入，收入带来投资本金
3. **阅读即投资**：每天阅读是在给自己做复利
4. **建立能力圈**：越早建立能力圈，复利效果越大

## 真实案例
- 芒格20多岁时做律师，同时大量阅读建立思维模型
- 他和巴菲特早期的投资收益来自他们多年积累的认知

## 常见错误
| 错误 | 表现 |
|------|------|
| 只存钱不投资自己 | 省钱但不学习 |
| 只投资不提升收入 | 研究理财但不管主业 |
| 忽视能力圈 | 什么热门学什么，不专注

## 检查清单
- [ ] 我每月在学习上投入了多少时间？
- [ ] 我在建立还是消耗自己的能力圈？
- [ ] 我的认知在增长吗？

Q1：芒格认为20岁最值得投资的是什么？
Q2：阅读为什么是一种自我投资？
Q3：你当前在投资自己还是在消费自己？
Q4：自我投资和财务投资的关系是什么？
Q5：制定一个你的自我投资计划。
