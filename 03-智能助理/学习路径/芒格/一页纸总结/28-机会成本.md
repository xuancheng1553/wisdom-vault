# 一页纸总结：机会成本

## 一句话定义
> 每做一件事，就放弃了做另一件事的机会。真正的成本不是你花了多少钱，而是你没选的那个选项可能带来的价值。

## 核心概念
1. **看不见的成本**：机会成本是你看不见的，但它是真实存在的
2. **时间是最贵的机会成本**：时间花在哪里，价值就在哪里
3. **比较选择**：做任何决策时，问自己「如果不做这个，我还能做什么？」
4. **机会成本随选择增加**：当你有更多选择时，机会成本更高
5. **不要被沉没成本影响**：已经花出去的钱不是成本，是沉没成本

## 真实案例
- 你花时间研究短线交易 → 机会成本是你本可以用这个时间阅读和学习的时间
- 芒格选择做投资而不是继续做律师 → 他放弃了律师的收入，但获得了更大的投资回报
- 你选择做资产配置群而不是单独炒股 → 放弃了个股收益的可能，但获得了更稳定的商业模式
- 花时间在小事上纠结 → 机会成本是这些时间可以用在大事上

## 常见错误
| 错误 | 表现 |
|------|------|
| 不考虑机会成本 | 只看显性成本，忽视隐形机会 |
| 低估时间成本 | 花大量时间省小钱 |
| 被沉没成本影响 | 因为已经投入了，所以继续投入 |
| 选择困难症 | 把时间花在比较上，不行动 |

## 检查清单
- [ ] 做这个选择我放弃了什么？
- [ ] 我花的时间有更好的用途吗？
- [ ] 我有没有被沉没成本影响？
- [ ] 如果现在重新选择，我还会做同样的决定吗？
