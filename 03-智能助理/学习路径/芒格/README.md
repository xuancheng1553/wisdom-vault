# 查理·芒格学习路径：5级进阶体系

## Level 1：认知入门
> 目标：了解芒格是谁、他为什么重要、核心思想地图

| 维度 | 内容 |
|------|------|
| 核心认知 | 芒格不是普通投资者，而是跨学科思维方法的集大成者 |
| 思想地图 | 多元思维模型、误判心理学、能力圈、反过来想、双轨分析、Lollapalooza效应 |
| 阅读起点 | 《穷查理宝典》前两章 |

## Level 2：方法掌握
> 目标：掌握芒格的核心方法论框架

- 多元思维模型的搭建方法
- 25个误判心理学的识别和规避
- 能力圈的划定和拓展方法
- 反过来想的逆向思维
- 双轨分析的并行运用

## Level 3：实践应用
> 目标：把芒格方法应用到投资、创业、管理、决策

- **投资**：检查清单、能力圈、反过来想
- **创业**：护城河分析、激励机制设计
- **管理**：选人用人、激励机制
- **决策**：双轨分析、多元思维模型

## Level 4：系统思维
> 目标：建立芒格格栅理论体系

- 跨学科模型的连接和交叉验证
- Lollapalooza效应的识别和利用
- 多个模型指向同一个结论时的信心

## Level 5：融会贯通
> 目标：将芒格智慧与其他智慧体系交叉对比

- 芒格×毛主席：实践论与多元思维模型、实事求是与能力圈
- 芒格×富兰克林：十三条美德与双轨分析
- 芒格×老子：反过来想与矛盾转化
